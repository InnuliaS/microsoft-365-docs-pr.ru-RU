<span data-ttu-id="73c4b-106">Пользователи с разрешениями для почтового ящика группы "Отправить как" или "Отправить от имени" могут отправлять сообщения с электронного адреса почтового ящика, если это разрешено администратором.</span><span class="sxs-lookup"><span data-stu-id="73c4b-106">Users with permissions to the group mailbox can send as or send on behalf of the mailbox email address if the administrator has given that user permissions to do that.</span></span> <span data-ttu-id="73c4b-107">Это особенно полезно для справки и поддержки почтовых ящиков, так как пользователи могут отправлять электронную почту из "Contoso Support" или "Создание службы получения".</span><span class="sxs-lookup"><span data-stu-id="73c4b-107">This is particularly useful for help and support mailboxes because users can send emails from "Contoso Support" or "Building A Reception Desk.</span></span>

- <span data-ttu-id="73c4b-117">**Внешние пользователи:** Вы не можете предоставлять пользователям за пределами организации (например, с помощью учетной записи Gmail) доступ к общему почтовому ящику.</span><span class="sxs-lookup"><span data-stu-id="73c4b-117">**External users:** You can't give people outside your business (such as people with a Gmail account) access to your shared mailbox.</span></span> <span data-ttu-id="73c4b-118">Если вы хотите сделать это, подумайте о том, чтобы вместо этого создать группу для Outlook.</span><span class="sxs-lookup"><span data-stu-id="73c4b-118">If you want to do this, consider creating a group for Outlook instead.</span></span> <span data-ttu-id="73c4b-119">Чтобы узнать больше, ознакомьтесь со статьей [Создание группы Office 365 в центре администрирования](../create-groups/create-groups.md).</span><span class="sxs-lookup"><span data-stu-id="73c4b-119">To learn more, see [Create an Office 365 group in the admin center](../create-groups/create-groups.md).</span></span>

- <span data-ttu-id="73c4b-132">**Вход:** Общий почтовый ящик не предназначен для прямого входа с помощью связанной с ним учетной записи пользователя.</span><span class="sxs-lookup"><span data-stu-id="73c4b-132">**Signing in:** A shared mailbox is not intended for direct sign-in by its associated user account.</span></span> <span data-ttu-id="73c4b-133">Вы всегда можете заблокировать вход для учетной записи общего почтового ящика и сохранить его блокировку.</span><span class="sxs-lookup"><span data-stu-id="73c4b-133">You should always block sign-in for the shared mailbox account and keep it blocked.</span></span>

- <span data-ttu-id="73c4b-136">**Удаление сообщения:** К сожалению, вы не можете запретить пользователям удалять сообщения в общем почтовом ящике.</span><span class="sxs-lookup"><span data-stu-id="73c4b-136">**Message deletion:** Unfortunately, you can't prevent people from deleting messages in a shared mailbox.</span></span> <span data-ttu-id="73c4b-137">Единственным выходом из этой ситуации может стать создание группы Office 365, а не общего почтового ящика.</span><span class="sxs-lookup"><span data-stu-id="73c4b-137">The only way around this is to create an Office 365 Group instead of a shared mailbox.</span></span> <span data-ttu-id="73c4b-138">Группа в Outlook аналогична общему почтовому ящику.</span><span class="sxs-lookup"><span data-stu-id="73c4b-138">A Group in Outlook is like a shared mailbox.</span></span> <span data-ttu-id="73c4b-139">Сравнение этих двух способов приведено в разделе [Сравнение групп](../create-groups/compare-groups.md).</span><span class="sxs-lookup"><span data-stu-id="73c4b-139">For a comparison of the two, see [Compare Groups](../create-groups/compare-groups.md).</span></span> <span data-ttu-id="73c4b-140">Дополнительные сведения о группах приведены в статье дополнительные сведения [о группах](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2.aspx)</span><span class="sxs-lookup"><span data-stu-id="73c4b-140">To learn more about Groups, see [Learn more about Groups](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2.aspx)</span></span>
