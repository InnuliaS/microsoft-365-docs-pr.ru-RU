# <a name="assign-licenses-to-users"></a>Назначение лицензий пользователям

::: moniker range="o365-worldwide"

Вы можете назначать лицензии пользователям на странице **Активные пользователи** или **Лицензии**. Выбор метода зависит от того, нужно ли назначить лицензии продуктов определенным пользователям или назначить пользователям лицензии на определенные продукты.

> [!NOTE]
> Если вы не используете новый Центр администрирования Microsoft 365, можно включить его с помощью переключателя **Попробовать новый Центр администрирования**, расположенного в верхней части главной страницы.

## <a name="assign-licenses-to-users-on-the-licenses-page"></a>Назначение лицензий пользователям на странице "Лицензии"

С помощью страницы **Лицензии** вы можете назначать лицензии на определенный продукт для 20 пользователей. На странице **Лицензии** отображается список всех продуктов, на которые у вас есть подписки, а также общее количество лицензий для каждого продукта, количество назначенных и доступных лицензий.

1. В Центре администрирования выберите **Выставление счетов** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842264" target="_blank">Лицензии</a>.
2. Выберите продукт, для которого нужно назначить лицензии.
3. Нажмите **Назначение лицензий**.
4. В области **Назначение лицензий пользователям** начните вводить имя и выберите его в результатах, чтобы добавить в список. Вы можете добавить до 20 пользователей одновременно.
5. Выберите **Включение и отключение приложений и служб**, чтобы назначить или удалить доступ к определенным элементам.
6. После завершения нажмите **Назначить**, а затем — **Закрыть**.

При наличии конфликта отобразится сообщение о возникшей проблеме и способе ее устранения. Например, если вы выбрали лицензии, включающие конфликтующие службы, сообщение об ошибке предложит проверить службы, входящие в состав каждой лицензии, и повторить попытку.

Изменение приложений и служб, к которым у пользователя есть доступ:

1. Выберите строку с пользователем.
2. В области справа выберите приложения и службы, к которым нужно предоставить доступ. Или отмените выбор, чтобы удалить соответствующий доступ.
3. После завершения нажмите **Сохранить**, а затем — **Закрыть**.

::: moniker-end

## <a name="assign-licenses-to-multiple-users-on-the-active-users-page"></a>Назначение лицензий нескольким пользователям на странице "Активные пользователи"

::: moniker range="o365-worldwide"

1. В Центре администрирования откройте страницу **Пользователи ** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">Активные пользователи</a>.

2. Установите флажки рядом с именами пользователей, которым вы хотите назначить лицензии.

3. Нажмите вверху **Дополнительные параметры (...)** и выберите **Управление лицензиями на продукты**.

4. В области **Управление лицензиями на продукты** выберите **Добавить к имеющимся назначениям лицензий на продукты** \> **Далее**.

5. В области **Добавить к имеющимся продуктам** установите переключатель для лицензии, которую вы хотите назначить выбранным пользователям, в положение **Вкл.**.

    По умолчанию пользователям автоматически назначаются все службы, связанные с этими лицензиями. Вы можете ограничить доступные для пользователей службы. Для служб, которых не должно быть у пользователей, установите переключатель в положение **Выкл**.

6. Внизу области выберите **Добавить** \> **Закрыть**.  

::: moniker-end

::: moniker range="o365-germany"

2. Установите флажки рядом с именами пользователей, которым вы хотите назначить лицензии.

3. В области **Массовые действия** выберите **Изменить лицензии продуктов**.

4. В области **Назначить продукты** выберите **Добавить к имеющимся назначениям лицензий на продукты** \> **Далее**.

5. Установите переключатель для лицензий, которые вы хотите назначить выбранным пользователям, в положение **Вкл**.

    По умолчанию пользователям автоматически назначаются все службы, связанные с этими лицензиями. Вы можете ограничить доступные для пользователей службы. Для служб, которых не должно быть у пользователей, установите переключатель в положение **Выкл**.

6. В нижней части области **Добавить к имеющимся продуктам** выберите **Добавить** \> **Закрыть** \> **Закрыть**.

## <a name="assign-licenses-to-one-user-on-the-active-users-page"></a>Назначение лицензий одному пользователю на странице "Активные пользователи"

::: moniker range="o365-worldwide"

1. В Центре администрирования откройте страницу **Пользователи ** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">Активные пользователи</a>.

2. На странице **Активные пользователи** выберите строку пользователя, которому вы хотите назначить лицензию.

3. В области справа выберите **Лицензии и приложения**.

4. Разверните раздел **Лицензии**, установите флажки для лицензий, которые нужно назначить, и нажмите **Сохранить изменения**.

::: moniker-end

4. В области **Лицензии на продукты** установите выключатель для лицензии, которую вы хотите назначить, в положение **Вкл.**

    По умолчанию пользователю автоматически назначаются все службы, связанные с этой лицензией. Вы можете ограничить доступные для пользователя службы. Для служб, которых не должно быть у пользователя, установите переключатель в положение **Выкл**.

5. В нижней части области **Лицензии на продукты** последовательно нажмите **Сохранить** \> **Закрыть** \> **Закрыть**.

## <a name="move-users-to-a-different-subscription"></a>Перевод пользователей на другую подписку

При наличии нескольких подписок вы можете перевести пользователей, которым уже назначена лицензия на одну подписку, на другую подписку, заменив существующую лицензию на другую.

::: moniker range="o365-worldwide"

1. В Центре администрирования откройте страницу **Пользователи ** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=834822" target="_blank">Активные пользователи</a>.

2. Установите флажки рядом с именами пользователей, для которых необходимо заменить лицензии.

3. Нажмите вверху **Дополнительные параметры (...)** и выберите **Управление лицензиями на продукты**.

4. В области **Управление лицензиями на продукты** выберите **Заменить имеющиеся назначения лицензий на продукты** \> **Далее**.

5. Установите переключатель для лицензий, которые вы хотите назначить этим пользователям, в положение **Вкл**.

    Вы можете ограничить доступные для пользователей службы. Для служб, которых не должно быть у этих пользователей, установите переключатель в положение **Выкл**. Все предыдущие назначения лицензий для выбранных пользователей удаляются.

6. В нижней части области **Заменить имеющиеся продукты** нажмите **Заменить** \> **Закрыть**.

::: moniker-end

::: moniker range="o365-germany"

1. В Центре администрирования откройте страницу **Пользователи** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=847686" target="_blank">Активные пользователи</a>.

2. Установите флажки рядом с именами пользователей, для которых необходимо заменить лицензии.

3. В области **Массовые действия** выберите **Изменить лицензии продуктов**.

4. В области **Назначить продукты** выберите **Заменить имеющиеся назначения лицензий на продукты** \> **Далее**.

5. Установите переключатель для лицензий, которые вы хотите назначить этим пользователям, в положение **Вкл**.

    Вы можете ограничить доступные для пользователей службы. Для служб, которых не должно быть у пользователей, установите переключатель в положение **Выкл**. Все предыдущие назначения лицензий для выбранных пользователей удаляются.

6. В нижней части области **Заменить имеющиеся продукты** нажмите **Заменить** \> **Закрыть** \> **Закрыть**.

::: moniker-end

## <a name="what-you-need-to-know-about-assigning-licenses-to-users"></a>Что необходимо знать о назначении лицензий пользователям

- Необходимо иметь права глобального администратора, администратора выставления счетов, администратора лицензий или администратора управления пользователями. Дополнительные сведения см. в статье [Роли администраторов в Office 365](../add-users/about-admin-roles.md).

- Вы можете [назначать лицензии учетным записям пользователей с помощью Office 365 PowerShell](https://go.microsoft.com/fwlink/p/?linkid=850410).

- Чтобы назначить лицензию существующей учетной записи пользователя, выполните указанные ниже действия. [Узнайте, как одновременно добавить учетную запись пользователя и назначить лицензию](../add-users/add-users.md).

- Некоторые службы, такие как Sway, назначаются пользователям автоматически и не требуют отдельного назначения.
