# <a name="cortana-in-microsoft-365"></a>Кортана в Microsoft 365

Кортана, ваш персональный помощник по работе, предоставляет возможности для экономии времени, а также уделять внимание большей важности. Кортана поможет вашим пользователям повысить производительность труда в течение всего дня и на рабочем, и в жизни. При входе с использованием действительных рабочих или учебных учетных записей пользователи могут получать облачные службы Кортаны в средствах повышения производительности Microsoft 365. Кортана в Microsoft 365 соответствует требованиям Microsoft 365 на уровне предприятия, безопасности и соответствия требованиям.

В качестве персонального помощника по повышению производительности Кортана разработана для обеспечения надежной и безопасной обработки и причин последующего использования данных Microsoft 365, таких как сообщения электронной почты, файлы, беседы и т. д., для экономии времени, повышения эффективности и повышения производительности пользователей.

При переносе вперед мы сосредоточены на Кортане в масштабах предприятия.

- Согласованность с другими службами Microsoft 365, Кортана в Microsoft 365 соответствует той же политике конфиденциальности, безопасности и соответствия требованиям уровня предприятия, которая отражается в [терминах веб-служб (OST)](https://www.microsoft.com/en-us/licensing/product-licensing/products).

- Новые службы помощи Кортаны на основе облачных служб, такие как информационные сообщения и воспроизводимые сообщения электронной почты, будут включены с помощью Кортаны в Microsoft 365 и полностью соответствуют этим обещаниям. В настоящее время эти функции доступны на английском (США) рынке для пользователей с почтовыми ящиками в компании US GEO, а также для сотрудников с несколькими национальными национальными почтовыми ящиками, основанными на US.

- Пользователи могут подключаться к Кортане в службах Microsoft 365, описанные в этой статье, с помощью клиентских приложений, таких как Outlook Mobile, в зависимости от условий лицензирования.

- Существующие возможности для потребителей, в том числе Кортана в Windows 10 (версия 1909 и более ранние версии) и приложение Кортана на iOS и Android, управляются [соглашением об использовании служб Майкрософт](https://www.microsoft.com/en-us/licensing/product-licensing/products) и [заявлением о конфиденциальности Майкрософт](https://privacy.microsoft.com/en-us/privacystatement). Эти термины также будут управлять Кортаной в службах Microsoft 365, предоставленных пользователю при входе с использованием учетных данных клиента.

## <a name="what-data-is-processed-by-cortana-in-microsoft-365"></a>Какие данные обрабатываются Кортаной в Microsoft 365?

Кортана в Microsoft 365 обрабатывает запросы от пользователя, данные Office, необходимые для выполнения запроса пользователя, а также другие телеметрии, создаваемые системами Microsoft для запуска службы. Данные, собираемые Кортаной в Microsoft 365, включают текстовое представление голосовых запросов пользователя (например, транскрипций из распознавания речи). Это текстовые данные, которые управляются в соответствии с [условиями веб-служб](https://www.microsoft.com/en-us/licensing/product-licensing/products). Он используется только для разработки и усовершенствования моделей машинного обучения, совместимых с [терминами веб-служб](https://www.microsoft.com/en-us/licensing/product-licensing/products).

## <a name="what-is-the-governance-model-for-customer-data-in-cortana-in-microsoft-365"></a>Что такое модель управления для данных клиентов в Кортане в Microsoft 365?

Согласованность с другими службами Microsoft 365 и Microsoft 365, Кортана в Microsoft 365 защищена и относится к [терминам веб-служб](https://www.microsoft.com/en-us/licensing/product-licensing/products). Это включает набор обещаний для защиты данных клиентов от случайных потерь, изменения, несанкционированного разглашения, доступа или уничтожения незаконному. Кроме того, для данных клиентов действуют ограничения на ограниченный доступ. Корпорация Майкрософт использует данные клиентов только для предоставления согласованных служб, а также для целей, совместимых с этими службами. В приведенной ниже таблице содержится подробная информация.

## <a name="cortana-in-microsoft-365-features"></a>Кортана в компонентах Microsoft 365

### <a name="play-my-emails"></a>Проигрывать мои сообщения

Воспроизведение писем (как подключенных через Outlook Mobile) — это управляемая голосовым интерфейсом, с помощью которой пользователи могут прослушивать свои папки "Входящие" с помощью динамиков на телефонном, наушникных и подключенных звуковых устройствах. Пользователи могут попросить Кортане читать последние сообщения электронной почты вслух и попросить Кортане выполнить такие действия, как помечать, архивировать, удалять и пропускать сообщения. Эта функция особенно полезна для отслеживания сообщений электронной почты во время работы или в дороге. Когда пользователь обращается к Кортане из воспроизведения электронных писем, запрос голосового голоса передается непосредственно в Кортану в Microsoft 365. Текстовая блок-речь для электронной почты пользователя обрабатывается в облаке Microsoft 365. Во время этого процесса никакие данные Microsoft 365 не обрабатываются на мобильном устройстве пользователя, а новые данные электронной почты не сохраняются. Транскрипция голосовых команд (например, "пометить как прочтенный", "Далее", "" ") может быть сохранена в соответствии с условиями защиты данных в [терминах веб-служб](https://www.microsoft.com/en-us/licensing/product-licensing/products).

Кортана будет вызываться, когда электронная почта защищена и ненадолго задерживается, прежде чем считывать сообщение, чтобы предоставить пользователям достаточно времени для приостановки воспроизведения или перехода к следующему сообщению. Как и для частного звонка, пользователи должны соблюдать осторожность при инициации воспроизведения в местах, где потенциально могут передаваться конфиденциальные данные. В этих случаях рекомендуется, чтобы сотрудники вашей организации наизноси наушники в соответствующих средах при воспроизведении почты в Outlook Mobile.

### <a name="opt-out-of-play-my-emails"></a>Отказ от проигрывания моих сообщений электронной почты

Пользователи могут отказаться от проигрывания моих сообщений, выполнив указанные ниже действия.

1. Откройте Outlook Mobile.

2. Перейдите в раздел **Параметры**.
  
3. Выберите пункт **проигрывать мои сообщения**.

4. Установите переключатель в положение отключено для учетных записей, которые требуется отключить.

### <a name="briefing-email"></a>Информационные сообщения

Кортана отправляет персонализированное информационное сообщение с помощью задач и обязательств, которые вы сделали с помощью удобного способа пометить их как **Завершенные** или запланировать время фокусировки для их выполнения. Кроме того, она включает в себя сводную информацию о собраниях и соответствующих документах в течение вашего дня. Кортана извлекает сведения из сообщений пользователя Microsoft 365 и сохраняет их в своем почтовом ящике Microsoft 365 до тех пор, пока они не будут объединены в информационные сообщения. Нет персональных данных, доступных за прев вашем почтовом ящике Microsoft 365.

### <a name="how-to-opt-out-of-briefing-email"></a>Как отказаться от использования информационного сообщения

Администраторы могут настроить краткую организацию для своей организации с помощью [переключателя](https://docs.microsoft.com/briefing/be-admin) в центре администрирования Microsoft 365. Пользователи могут отказаться от использования информационного сообщения Кортаны, выбрав пункт отказ от подписки в нижнем колонтитуле сообщения.

Мы будем продолжать внедрять дополнительные Кортана в функции Microsoft 365, чтобы повысить производительность организации.
