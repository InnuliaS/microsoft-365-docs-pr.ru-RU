# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Получение последней версии аналитики использования Microsoft 365

Приложение шаблона можно обновить с использованием новых данных или новых зрительных образов несколько раз в год. Существующий экземпляр будет продолжать работать, но если вы хотите получить последнюю версию, необходимо создать новый экземпляр и применить к новому экземпляру все настройки. Сведения о [включении аналитики использования Microsoft 365](enable-usage-analytics.md).
