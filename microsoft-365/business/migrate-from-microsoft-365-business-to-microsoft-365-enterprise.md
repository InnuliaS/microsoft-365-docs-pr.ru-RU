# <a name="migrate-from-microsoft-365-business-to-microsoft-365-enterprise-e3"></a>Переход с Microsoft 365 бизнес на Microsoft 365 корпоративный E3

Microsoft 365 Business имеет все необходимое для малого бизнеса, объединяя лучшие облачные приложения для продуктивной работы с помощью простого управления устройствами и обеспечения безопасности, которые позволяют сотрудникам выполнять свои действия. Однако в некоторых случаях вам может потребоваться перенести Microsoft 365 бизнес-подписку на Microsoft 365 корпоративный. 

Например, ваша организация увеличилась и нуждается в более чем 300 лицензиях (Поздравляем, кстати).

Кроме того, для вашего бизнеса требуются корпоративные функции, такие как Office 365 профессиональный плюс, Windows 10 Корпоративная или корпоративные клиентские лицензии (CAL).

Обновление упрощается: вы можете запустить обновление [из центра администрирования](https://docs.microsoft.com/office365/admin/subscriptions-and-billing/upgrade-to-different-plan?view=o365-worldwide). Все ваши данные и конфигурация в текущей подписке поддерживаются. Вы не можете подготовиться к миграции и не предпринимать никаких дополнительных действий, кроме использования новых функций. 

>[!Note]
>Вы также можете использовать подписку на Microsoft 365 Business для получения до 300 рабочих мест и получения подписки на Microsoft 365 корпоративный E3 для более чем 300 рабочих мест. Однако Office 365 ATP не входит в состав Microsoft 365 корпоративный E3. Для продолжения защиты от угроз необходимо добавить дополнительные лицензии Office 365 ATP, чтобы обеспечить лицензирование всех пользователей в области политик Office 365 ATP.
>

## <a name="migration"></a>Миграция

Чтобы выполнить миграцию, свяжитесь с партнером, чтобы перенести Microsoft 365 бизнес-подписку и лицензии на подходящую подписку Microsoft 365 корпоративный E3 с лицензиями.

В следующих разделах описываются изменения, которые необходимо выполнить, если они есть, и действия, которые можно выполнить после миграции.

### <a name="microsoft-365-subscription-configuration-and-data"></a>Конфигурация и данные подписки Microsoft 365

Вам не нужно вносить никаких изменений в текущую подписку или данные перед миграцией, включая следующие:

- Конфигурация подписки, например доменные имена DNS.
- Учетные записи пользователей и групп и параметры проверки подлинности, такие как многофакторная проверка подлинности или политики условного доступа.
- Конфигурации служб производительности и их данные, такие как Teams, почтовые ящики Exchange Online, сайты SharePoint Online, папки OneDrive для бизнеса и записные книжки OneNote.

Теперь пользователи могут работать с неограниченным хранилищем в папках почтовых ящиков Exchange Online и OneDrive для бизнеса.

Вы можете начать использовать обнаружение облачных приложений, работоспособность Azure AD Connect и единый вход для более чем 10 приложений.

>[!Note]
>Пользователи, перенесенные в Microsoft 365 корпоративный E3, больше не могут использовать Outlook Customer Manager и Милеик.
>

<a name="threat-protection"></a>
### <a name="threat-protection"></a>Защита от угроз

Windows 10 бизнес включает следующие средства защиты:

- Обеспечение целостности процесса загрузки операционной системы
- Обеспечение целостности важных рабочих компонентов
- Повышенная уязвимость и устранение уязвимостей с использованием нулевого дня
- Защита сети на основе репутации для Microsoft EDGE, Internet Explorer и Chrome
- Брандмауэр на основе узла
- Снижение опасности для атаки программой «шантажистом»
- Изоляция на основе оборудования для Microsoft Edge
- Управление приложениями на базе интеллектуального графа безопасности
- Управление устройством (USB)
- Защита сети от угроз для веб-угроз
- Правила предотвращения вторжения для узла

Windows 10 Корпоративная E3 также включает управление изоляцией на основе аппаратного обеспечения для Microsoft Edge.

>[!Note]
>Для всех пользователей, перенесенных в Microsoft 365 корпоративный E3, требуется лицензия на Office 365 ATP для продолжения защиты от угроз. Обязательно приобретайте дополнительные лицензии Office 365 ATP, чтобы обеспечить лицензирование всех пользователей в области политик Office 365 ATP. 
>

### <a name="device-management-with-intune"></a>Управление устройствами с помощью Intune

Нет необходимости вносить какие – либо изменения в текущую конфигурацию Intune перед миграцией, которая включает зарегистрированные устройства и параметры устройств и приложений.

### <a name="windows-10"></a>Windows 10

Microsoft 365 Business включает Windows 10 бизнес, которую можно установить с помощью Windows автопилота. При переходе на Microsoft 365 корпоративный E3 каждая пользовательская лицензия включает Windows 10 Корпоративная версия E3, которую можно также установить с помощью Windows автопилота.

<a name="office-365-business"></a>
### <a name="office-365-business"></a>Office 365 бизнес

Office 365 Business Client, установленный на устройствах, автоматически начнет использовать функции Office 365 профессиональный плюс. После миграции теперь можно использовать следующие компоненты:

 - Активация корпоративных лицензий с помощью групповой политики
 - Телеметрии приложений
 - Обновление элементов управления
 - Сравнение и запрос электронной таблицы
 - Бизнес-аналитика
