# <a name="microsoft-365-business-frequently-asked-questions"></a>Вопросы и ответы по Microsoft 365 Business

## <a name="general"></a>Общие

### <a name="what-is-microsoft-365-business"></a>Что такое Microsoft 365 бизнес? 
Microsoft 365 — это интегрированное решение, объединяющее лучшие средства повышения производительности, безопасность и возможности управления устройствами для малых и средних предприятий. Он включает следующее:

**Набор средств для бизнес-работы и совместной работы** 
* Word, Excel, PowerPoint, Outlook, OneNote, Publisher и Access 
* Exchange, OneDrive, Skype для бизнеса, Microsoft Teams и SharePoint. 
* Бизнес-приложения Office (резервирования, Милеик<sup>[1](#footnote1)</sup>). 
 
**Функции управления устройствами и обеспечения безопасности корпоративного уровня** 
- Обеспечивает защиту от внешних угроз, таких как фишинговые и сложные вредоносные программы с Office 365 Advanced Threat Protection Plan 1 и Guard в Защитнике Windows. 
-  Помогает управлять доступом к конфиденциальной информации и их передачей с помощью политик защиты от потери данных Office 365 и Azure Information Protection (план 1). 
-  Помогает защитить, сохранять и создавать резервные копии данных с помощью архивации на базе Exchange Online.  
-   Защита приложений для Office и других мобильных приложений с помощью средства защиты приложений Intune. 
-  Управление устройствами для компьютеров с Windows 10, MacOS и мобильных устройств с помощью управления устройствами Intune. 
-  Защита удостоверений с многофакторной проверкой подлинности, самостоятельного сброса пароля и условного доступа.
-  Согласованная конфигурация безопасности на&mdash;устройствах обеспечивает защиту данных компании на разных устройствах; Защитник Windows, который постоянно включен и находится в актуальном состоянии. 
 
 
**Упрощенное развертывание устройств и настройка пользователей** 
* Единая консоль администрирования для настройки пользователей и устройств и управления ими 
* Автоматическая установка приложений Office на компьютерах с Windows 10. 
* Всегда обновлены Office + Windows 10. 
* Упрощенное развертывание компьютеров с помощью Windows автопилота.

**Другие права** 
- Microsoft 365 Business Customers также имеют доступ к функциям активации виртуальных рабочих столов Windows и общего компьютера Office. Прочитайте [блог Microsoft 365 Business](https://techcommunity.microsoft.com/t5/Microsoft-365-Business-Blog/bg-p/Microsoft365BusinessBlog) , чтобы узнать больше. Описание также можно найти в [статье Microsoft 365 Business Service](https://docs.microsoft.com/office365/servicedescriptions/microsoft-365-service-descriptions/microsoft-365-business-service-description).

### <a name="who-should-consider-adopting-microsoft-365-business"></a>Кому целесообразно внедрять Microsoft 365 бизнес? 
Microsoft 365 Business — это комплексное облачное решение для обеспечения безопасности, которое позволяет:  
- Защита бизнеса от расширенных киберугрозами с помощью замысловатого фишинга и защиты от атак с помощью средства для атаки. 
- Контролируйте доступ к конфиденциальным сведениям с помощью шифрования, чтобы предотвратить случайное предоставление доступа к данным пользователям, не имеющим полномочий для просмотра. 
- Обеспечьте безопасность устройств, которые подключаются к Организации, и обеспечьте защиту данных на устройствах с iOS, Android, Windows и Mac, а также на обновленных устройствах Windows.

Microsoft 365 Business полностью интегрирована с Office 365, чтобы предоставить вам знакомое расположение для администрирования, выставление счетов и круглосуточная поддержка. Он состоит из технологии корпоративного уровня, созданной для предприятий с менее чем 300 сотрудниками.

Microsoft 365 Business можно приобрести у <a href="https://www.microsoft.com/solution-providers/search" target="_blank">партнера корпорации Майкрософт</a> или непосредственно у <a href="https://www.microsoft.com//microsoft-365/business" target="_blank">корпорации Майкрософт</a>. При принятии решения о покупке непосредственно у корпорации Майкрософт или партнера Майкрософт, рекомендуется оценить возможности персонала и желание поддерживать ИТ-инфраструктуру. Партнер Майкрософт может помочь в развертывании ИТ-инфраструктуры и управлении ею, включая решения Майкрософт.

При покупке через партнеров Майкрософт цены могут меняться в зависимости от предоставляемых партнером услуг и его модели ценообразования для Microsoft 365 бизнес. 

### <a name="is-there-a-cap-to-how-many-microsoft-365-business-seats-a-customer-can-have"></a>Есть ли ограничение на количество рабочих мест Microsoft 365 бизнес, которое может приобрести клиент? 
Microsoft 365 бизнес предназначался для малых и средних предприятий с минимальными и средними требованиями к сложности ИТ-сред. Клиенты могут приобрести до 300 лицензий Microsoft 365 бизнес для своей организации. Клиенты могут смешивать и сопоставлять облачные подписки. В результате клиенты могут добавлять корпоративные лицензии Microsoft 365 Enterprise на одну и ту же учетную запись. 
 
Когда клиенты считают среду, состоящую из нескольких типов подписки, они должны работать с доверенными ИТ-консультантами, чтобы определить, как лучше всего управлять и защищать различные подписки как на Microsoft 365 Business и Microsoft 365 корпоративный использование различные возможности для защиты приложений и данных и управления ими.

### <a name="can-i-combine-microsoft-365-business-with-other-microsoft-subscription-offerings"></a>Можно ли объединить Microsoft 365 бизнес с другими предложениями по подписки Майкрософт? 
Да, клиенты могут объединить свои подписки на Microsoft 365 бизнес с планами и надстройками Azure, Dynamics 365, Enterprise Mobility + Security и Office 365.

### <a name="is-everyone-in-my-business-required-to-have-a-microsoft-365-business-subscription"></a>Всем ли сотрудникам моей организации необходимо иметь подписку на Microsoft 365 бизнес? 
Нет, подписка на Microsoft 365 бизнес необходима не всем, однако преимущества управления и безопасности доступны только пользователям с устройствами, управляемыми с помощью подписки на Microsoft 365 бизнес. 
 
Стандартизация ИТ-среды способствует уменьшению затрат на обслуживание и безопасность с течением времени, а также о состоянии, которое должны стремиться к достижению предприятия. Однако мы понимаем, что некоторые пользователи малого и среднего размера обновляют свое программное обеспечение в основном при обновлении своего оборудования в течение длительного периода. Предприятия могут развертывать Microsoft 365 для бизнеса в рамках своей организации, но для обеспечения оптимальной защиты конфиденциальных бизнес-данных и единообразия взаимодействия рекомендуется развертывать все пользователи.

### <a name="how-can-i-know-if-the-hardware-and-software-i-run-today-is-compatible-with-microsoft-365-business"></a>Как понять, совместимо ли используемое мной в данный момент оборудование и программное обеспечение сегодня с Microsoft 365 бизнес? 
Если на вашем устройстве установлена операционная система Windows 7 Профессиональная или более поздней версии, она, скорее всего, соответствует минимальным требованиям для Microsoft 365 Business. Для некоторых функций Windows 10, таких как Кортана, Windows Hello и мультисенсорных, требуется определенное оборудование, которое доступно только на новых компьютерах. Дополнительные сведения приведены в разделе <a href="https://www.microsoft.com/windows/windows-10-specifications" target="_blank">требования к системе для Windows 10 Pro</a> . 
 
Ожидается, что большинство существующих классических приложений (Win32) будут поддерживаться в Windows 10 без каких-либо изменений. Клиентам и их доверенным ИТ-консультантам следует ознакомиться с рекомендуемым процессом тестирования приложений на <a href="https://docs.microsoft.com/windows/deployment/planning/windows-10-compatibility#recommended-application-testing-process" target="_blank">совместимость с Windows 10</a> и <a href="https://products.office.com/office-system-requirements#subscription-plans-section" target="_blank">системными требованиями для Office</a> для упрощения перехода на Microsoft 365 бизнес. 

### <a name="what-is-windows-10-business"></a>Что такое Windows 10 для бизнеса? 
Windows 10 бизнес — это набор облачных служб и возможностей управления устройствами, которые дополняют Windows 10 Pro и обеспечивают централизованное управление и управление безопасностью Microsoft 365 Business.

В Windows 10 бизнес также включены некоторые функции защитника Майкрософт, такие как контролируемый доступ к папкам и защита сети для веб-потоков. Кроме того, она включает службу Windows автопилота, которая упрощает развертывание новых компьютеров с Windows 10.

Если у вас есть устройства, лицензированные для Windows 7, 8 и 8,1 Professional, Microsoft 365 бизнес предоставляет обновление до Windows 10 Pro, которое является необходимым условием для развертывания Windows 10 Business.
> [!IMPORTANT]
> При обновлении необходимо указать исходный ключ продукта, в противном случае обновление не будет работать. 

### <a name="how-does-microsoft-365-business-help-support-our-bring-your-own-device-byod-policy"></a>Как Microsoft 365 бизнес обеспечивает поддержку политики использования собственных устройств (BYOD) в моей компании? 
Многие сотрудники предпочитают использовать свои собственные мобильные телефоны или планшеты для доступа к личной и рабочей информации вместо использования нескольких устройств для различных целей. Использование личных устройств для работы достаточно распространено, но повышает риск того, что коммерческая информация окажется в чужих руках. Многие конкурирующие мобильной решения для защиты данных требуют переключения устройства в определенный режим или использования другого сложного механизма, которого пользователя будут избегать по причине его сложности. 
 
Microsoft 365 Business предлагает пользователям простое, но мощное средство, позволяющее сотрудникам использовать персональные устройства для работы, в то же время предоставляя бизнес-возможности для того, чтобы эти устройства не могли получать доступ к бизнес-данным, сохранять их и/или предоставлять к ним общий доступ. Это обеспечивается следующими средствами. 
* **Защита приложений для приложений Office Mobile** помогает защитить данные Office включая электронную почту, календарь, контакты и документы на мобильных устройствах iOS и Android, предоставляя политики, такие как автоматическое удаление бизнес-данных после предопределенного периода времени, не требующего подключения к службе, необходимо, чтобы эти сведения сохранялись только в OneDrive для бизнеса, требуя проверки ПИН-кода или отпечатков пальцев для доступа к приложениям Office и предотвращения копирования данных компании из приложения Office в личные Управление мобильными приложениями для других мобильных приложений с помощью Intune также доступно для подписчиков Microsoft 365 Business. 
  
* **Управление устройствами для компьютеров с Windows 10** позволяет компаниям самостоятельно устанавливать и поддерживать такие возможности, как защита от вредоносных программ с помощью Защитника Windows, автоматические обновления и отключение экрана по истечении предписанного промежутка времени. Кроме того, потеряны или украденные устройства с Windows 10 могут быть полностью очищены от бизнес-приложений и данных с помощью центра администрирования. 

* **Управление устройствами для iOS, android & MacOS** Features помогает предприятиям безопасно управлять разнообразной экосистемой устройств, включающей устройства iOS, Android, Windows и MacOS. 

### <a name="how-does-microsoft-365-business-help-protect-pcs-in-my-organization-from-malicious-attacks"></a>Как Microsoft 365 бизнеса помогает защитить компьютеры в организации от вредоносных атак? 
ПК под управлением Microsoft 365 Business защищены защитником Windows, который является недоступным. 1 антивирусная функция в Windows 10 защищает больше компьютеров от вирусов, вредоносных программ, шпионских программ и других угроз, чем любое другое решение.  С помощью Microsoft 365 Business организации могут гарантировать, что защита защитника Windows запущена и всегда обновлена на всех устройствах с Windows 10.  
 
Windows 10 бизнес также включает в себя защитник Windows Defender, новый набор возможностей предотвращения вторжения. Один из его компонентов, контролируемого доступа к папкам, перестает отвечать на атаки, блокируя папки и предотвращая доступ несанкционированных приложений к важным файлам пользователя. 

### <a name="whats-the-difference-between-office-365-business-premium-microsoft-365-business-and-microsoft-365-enterprise"></a>В чем разница между Office 365 бизнес премиум, Microsoft 365 бизнес и Microsoft 365 Enterprise? 
У корпорации Майкрософт есть различные решения по управлению производительностью и обеспечению безопасности, которые могут учитываться при обновлении инфраструктуры настольных компьютеров и устройств, но при этом все функции и функции становятся более мощными. 

**Office 365 Business Premium** обеспечивает повышенную производительность приложений и служб Office 365, но не включает функции защиты приложений и управления устройствами в Microsoft 365 Business.

**Microsoft 365 Business** сочетает в себе приложения и службы Office 365 с расширенными возможностями безопасности для защиты бизнеса от расширенных киберугрозами, обеспечения безопасности данных и управления устройствами. Этот выпуск включает упрощенную консоль управления, посредством которой осуществляется администрирование политик устройств и данных. Большинство компаний малого и среднего размера можно использовать в Microsoft 365 Business. 
 
**Microsoft 365 корпоративный** это набор планов лицензирования, которые обеспечивают повышенный уровень соответствия требованиям и управления безопасностью в Microsoft 365 бизнес и предназначены для корпоративных клиентов и клиентов с более чем 300 пользователей. Кроме того, в планах Microsoft 365 Enterprise предусмотрены дополнительные функции, в том числе средства бизнес-аналитики и аналитики.

### <a name="can-i-switch-my-office-365-plan-to-microsoft-365-business"></a>Можно ли перейти с плана Office 365 на Microsoft 365 бизнес? 
Да, пользователи могут изменить свой план с соответствующего условиям плана Office 365 на Microsoft 365 бизнес. В зависимости от текущего плана клиента это может привести к снижению или увеличению ежемесячных расходов.

### <a name="in-what-regions-is-microsoft-365-business-available"></a>В каких регионах доступен Microsoft 365 бизнес? 
Microsoft 365 Business доступна всем партнерам и клиентам, на которых доступно приложение Office 365. Список <a href="https://products.office.com/business/international-availability" target="_blank">языков, стран и регионов</a>, доступных в Office 365. 

### <a name="is-there-a-microsoft-365-business-trial-i-may-use-to-evaluate-the-offer"></a>Существует ли пробная версия Microsoft 365 бизнес для оценки этого решения? 
Пробная версия Microsoft 365 Business доступна для поставщиков служб шифрования. Пробная версия для прямых клиентов будет доступна позже. 
 
### <a name="what-should-customers-and-partners-know-before-running-microsoft-365-business-within-their-organization"></a>Что клиентам и партнерам следует знать перед запуском Microsoft 365 бизнес в организации? 
Клиенты, желающие работать с полноценными возможностями Microsoft 365 Business, должны работать под управлением Windows 7, 8,1 или 10 Pro<sup>[2](#footnote2)</sup> на существующих рабочих станциях. На существующих компьютерах с Windows 10 Pro должно быть установлено обновление Creators Update. 

## <a name="deployment"></a>Развертывание

### <a name="what-should-customers-consider-when-planning-a-microsoft-365-business-deployment"></a>Что следует принять во внимание клиентам при планировании развертывания Microsoft 365 бизнес? 
Самый прямой путь к успешному развертыванию Microsoft 365 бизнес — это взаимодействие с партнером корпорации Майкрософт. Партнеры прошли обширную подготовку и обладают большим опытом работы с широким спектром пользовательских сценариев и лучше всего понимают ваши потребности и особенности вашей среды. Клиенты, имеющие в штате опытных ИТ-сотрудников, могут использовать раздел <a href="https://support.office.com/article/496e690b-b75d-4ff5-bf34-cc32905d0364" target="_blank">Начало работы с Microsoft 365 бизнес</a> для помощи в развертывания Microsoft 365 бизнес. 

### <a name="does-microsoft-365-business-include-the-full-capabilities-of-microsoft-intune"></a>Включает ли Microsoft 365 бизнес все возможности Microsoft Intune? 
Да, Microsoft 365 Business подписчики лицензированы для использования полных возможностей Intune для iOS, Android, MacOS и другого управления устройствами на разных платформах. Функции, недоступные в упрощенной консоли управления в Microsoft 365 Business, такие как управление сторонними приложениями и настройка профилей Wi-Fi, VPN-сертификаты, можно управлять в полной консоли Intune. 
  

### <a name="does-azure-active-directory-premium-p1-come-with-microsoft-365-business"></a>Поступили ли Azure Active Directory Premium P1 с Microsoft 365 Бизнес? 
Microsoft 365 бизнес включает в себя выбор таких функций Azure AD Premium P1 (ААДП P1), как самостоятельный сброс паролей с обратной записью в AD, Azure MFA и условный доступ. Он не включает в себя полностью ААДП P1. Дополнительные сведения см. в [описании Microsoft 365 Business Service](https://technet.microsoft.com/library/b042d1fd-c997-46d8-8c02-1df4c932f148).
 
 
### <a name="does-microsoft-365-business-allow-customers-to-manage-macs"></a>Позволяет ли Microsoft 365 бизнес управлять компьютерами Mac? 
Intune помогает безопасно управлять устройствами iOS, Android, Windows и MacOS. 
 
### <a name="what-is-windows-autopilot"></a>Что такое Windows AutoPilot? 
Windows AutoPilot — это служба, которая упрощает развертывание новых компьютеров с Windows 10. Этот процесс можно выполнить, когда конечный пользователь в первый раз&mdash;входит в систему Microsoft 365 Business, не обращаясь к устройству&mdash;с помощью централизованного управления средствами управления Microsoft 365 бизнес. Можно также использовать Windows AutoPilot для имеющихся компьютеров с Windows 10 Профессиональная Creators Update (или более поздней версией), для которых был выполнено восстановление до заводских настроек. Подробные сведения об автопилоте Windows можно найти в <a href="https://blogs.technet.microsoft.com/windowsitpro/2017/06/29/modernizing-windows-deployment-with-windows-autopilot/" target="_blank">этой записи блога 2017 июня</a>.

## <a name="compatibility"></a>Совместимость

### <a name="can-i-add-office-365-add-ons-to-microsoft-365-business"></a>Можно ли добавить надстройки Office 365 в Microsoft 365 бизнес? 
Все надстройки, которые могут быть добавлены в Office 365 бизнес премиум, можно добавить в Microsoft 365 бизнес. Это означает, что вы можете приобрести Office 365 Cloud App Security, расширенное соответствие требованиям, логика операций с угрозами, MyAnalytics, PowerBI Pro и голосовые конференции. 
 
### <a name="can-i-add-phone-system-and-calling-plans-to-microsoft-365-business"></a>Можно ли добавить планы "Телефонная система" и "Вызовы" в Microsoft 365 бизнес? 
Нет, эти планы зарезервированы для пользователей с более сложными потребностями. Клиентам, которым необходимы данные возможности, рекомендуется ознакомиться с предложениями Microsoft 365 корпоративный. 
 
### <a name="can-microsoft-365-business-customers-use-windows-defender-advanced-threat-protection"></a>Могут ли клиенты Microsoft 365 бизнес использовать службу Advanced Threat Protection в Защитнике Windows? 
Нет, пользователям, которым необходима служба Advanced Threat Protection в Защитнике Windows, потребуется Windows 10 Корпоративная E5 или Microsoft 365 корпоративный E5. 

## <a name="partner-opportunity"></a>Возможности для партнеров

### <a name="where-can-i-learn-more-about-the-opportunities-and-benefits-in-becoming-a-microsoft-partner"></a>Где можно узнать подробнее о возможностях и преимуществах получения статуса партнера Майкрософт? 
Поставщики ИТ-услуг, которые еще не являются партнерами Майкрософт, могут получить дополнительные сведения о программе поставщиков облачных решений (Майкрософт) на странице [https://partners.office.com/microsoft365business](https://partners.office.com/microsoft365business).

### <a name="where-can-i-learn-how-to-sell-microsoft-365-business"></a>Где можно узнать, как продавать Microsoft 365 бизнес? 
Партнеры, которые в данный момент занимаются продажами Office 365, могут использовать те же методы консультационных продаж для продажи Microsoft 365 бизнес. Кроме того, мы представляем дополнительные ресурсы и учебные курсы для вашей группы по продажам, чтобы изучить существующую среду для настольных ПК, требования к Active Directory, мобильность и обеспечение безопасности, чтобы обеспечить эффективное взаимодействие со всеми преимуществами Microsoft 365 Бизнес — это способ, относящийся к клиенту. Эти ресурсы можно найти на портале Office для партнеров по адресу [https://partners.office.com/microsoft365business](https://partners.office.com/microsoft365business). 

### <a name="how-can-microsoft-365-business-help-partners-increase-the-profitability"></a>Как Microsoft 365 бизнес может помочь партнерам повысить рентабельность? 
Microsoft 365 бизнес поможет партнерам снизить затраты путем повышения операционной эффективности и выручки за счет продажи дополнительных услуг. Исследование Forrester Research "Общий экономический эффект Microsoft 365 бизнес", проведенное в июне 2017 года (https://www.microsoft.com/microsoft-365/partners/PartnerTEI), показывает, что Microsoft 365 бизнес положительно скажется на прибыльности партнеров. 

В рамках исследования партнеры сообщили, что благодаря Microsoft 365 бизнеса они достичь следующих показателей. 

- Увеличение выручки за \[разовые\] услуги развертывания и консультационные услуги на 20 процентных пунктов
- Увеличение коэффициента сопутствующих продаж управляемых услуг на 10 процентных пунктов
- Увеличение рентабельности консультационных услуг и \[регулярных\] услуг управления на 8 процентных пунктов (за счет снижения расходов)

### <a name="what-resources-are-available-to-partners-to-sell-deploy-and-support-microsoft-365-business"></a>Какие ресурсы доступны для партнерам для продажи, развертывания и поддержки Microsoft 365 бизнес?
Корпорация Майкрософт предоставляет широкий выбор ресурсов для партнеров CSP для продажи и поддержки Microsoft 365 бизнес. Их можно найти на странице [https://partners.office.com/microsoft365business](https://partners.office.com/microsoft365business).

### <a name="what-up-sell-opportunities-does-microsoft-365-business-give-partners"></a>Какие возможности для увеличения объема продаж предоставляет партнерам Microsoft 365 бизнес? 
Microsoft 365 бизнес позволяет партнерам поддерживать свои позиции доверенного помощника с клиентами, создавая твердую и надежную платформу для продажи дополнительных служб и обновления существующих продуктов и служб. Microsoft 365 Business предоставляет возможность получить обсуждение обновления с клиентами, которые теперь используют Exchange Server, Exchange Online или Office 365 бизнес Essentials. Кроме того, партнеры могут получить больше дохода от повышенной управляемой службы и/или вознаграждений за техническую поддержку. 
 
С помощью новой функции автопилота Windows, входящей в состав Microsoft 365 Business, партнеров, которые были релуктант для продажи новых устройств с ОС Windows, так как логистика и затраты на развертывание могут оказаться гораздо более привлекательными. Клиенты, которые могут быть уверены в безопасности своих локальных и мобильных устройств, также скорее всего будут вкладываться в дополнительные службы, например Dynamics 365. 
 
### <a name="should-partners-sell-microsoft-365-business-over-other-plans-from-microsoft"></a>Следует ли партнерам продавать Microsoft 365 бизнес через другие планы корпорации Майкрософт? 
Поставщику облачных решений Майкрософт всегда следует продавать тот план, который оптимально соответствует потребностям и бюджету клиента. Например, если клиент должен отвечать требованиям, предъявляемым к конфиденциальности и безопасности, CSP может продавать Microsoft 365 бизнес и все надстройки, которые помогают клиенту выполнить требования, или предложить расширенную безопасность и управление, предоставляемые корпорацией Майкрософт 365 Business SKU Enterprise. 
 
### <a name="some-of-my-customers-have-devices-that-arent-genuine-will-microsoft-365-business-make-these-devices-genuine"></a>Некоторые из моих клиентов имеют устройства, которые не являются подлинными; подлинность этих устройств будет возложить Корпорация Майкрософт 365? 
Microsoft 365 бизнес не делает подлинной подлинной версии Windows без подлинности. Microsoft 365 Business обеспечивает возможность обновления до последней подлинной версии Windows 10 Pro, позволяющей пользователям подлинной системы Windows 7, 8 или 8,1 Pro выполнить обновление до последней подлинной версии Windows 10 Pro. 

### <a name="what-is-the-gdpr-and-how-does-microsoft-365-business-help-customers-with-their-compliance-obligations"></a>Что такое GDPR и как Microsoft 365 бизнес помогает клиентам выполнять свои обязательства по соблюдению нормативных требований?
Общие правила защиты данных (GDPR) — это совершенно новый регламент о конфиденциальности, который предоставляет гражданам Европейского Союза (ЕС) возможность лучше контролировать свои персональные данные, а также требует от организаций обеспечить защиту этих персональных данных. Для GDPR требуются Организации, которые контролируют и обрабатывают персональные данные, связанные со странами Европейского союза, только с помощью сторонних обработчиков данных, которые отвечают требованиям GDPR для обработки персональных данных. В марте 2017 года корпорация Майкрософт предоставила возможность следующих договорных гарантий. Пользователи, у которых есть вопросы по поводу того, как корпорация Майкрософт может помочь им удовлетворить свои дополнительные обязательства по GDPR, должны узнать о расширенных функциях обеспечения соответствия требованиям и безопасности, доступных в Microsoft 365 Business (например, Azure Information Protection, Data Защита от потери данных, Advanced Threat Protection и т. д.), а также в других наборах (например, Microsoft 365 Enterprise и т. д.). Для получения дополнительных сведений посетите страницу [www.microsoft.com/gdpr](https://www.microsoft.com/gdpr).
