# <a name="microsoft-365-business-frequently-asked-questions"></a>Вопросы и ответы по Microsoft 365 бизнес

## <a name="general"></a>Общие

### <a name="what-is-microsoft-365-business"></a>Что такое Microsoft 365 бизнес? 
Microsoft 365 — это интегрированное решение, объединяющее лучшие средства повышения производительности, безопасность и возможности управления устройствами для малых и средних предприятий. 

**Комплексный набор средств для совместной работы и повышения производительности для бизнеса** 
* Word, Excel, PowerPoint, Outlook, OneNote, Publisher и Access 
* Exchange, OneDrive, Skype для бизнеса, Microsoft Teams, SharePoint 
* Бизнес-приложения Office (Bookings, Outlook Customer Manager, MileIQ<sup>[1](#footnote1)</sup>, Microsoft Listings<sup>[1](#footnote1)</sup>, Microsoft Connections<sup>[1](#footnote1)</sup>, Microsoft Invoicing<sup>[1](#footnote1)</sup>) 
 
**Функции управления устройствами и обеспечения безопасности корпоративного уровня** 
- Обеспечивает защиту от внешних угроз, таких как фишинговые вредоносные программы _Амп_ с помощью Office 365 Advanced Threat Protection и Guard в Защитнике Windows 
-  Помогает управлять _Амп_ управления доступом к конфиденциальным сведениям и их передачей с помощью политик защиты от потери данных в Office 365 (доступно в летом, 2018) и Azure Information Protection (план 1) 
-  Помогает защитить, сохранять и создавать резервные копии данных с помощью архивации на базе Exchange Online  
-   Защита приложений для Office _Амп_ другие мобильные приложения с защитой приложений Intune 
-  Управление устройствами для компьютеров с Windows 10, MacOS _Амп_ мобильных устройств с управлением устройствами Intune 
-  Единая конфигурация безопасности для устройств — защита данных компании между устройствами — защитник Windows, всегда включено и обновлен 
 
 
**Упрощенное развертывание устройств и настройка пользователей** 
* Единая консоль администрирования для настройки пользователей и устройств и управления ими 
* Автоматическая установка приложений Office на компьютерах с Windows 10 
* Регулярное обновление Office + Windows 10 
* Упрощенное развертывание компьютеров с помощью Windows AutoPilot

### <a name="who-should-consider-adopting-microsoft-365-business"></a>Кому целесообразно внедрять Microsoft 365 бизнес? 
Microsoft 365 бизнес — это решение, созданное для малых и средних заказчиков с практически отсутствующими ИТ-ресурсами в штате, которое объединяет лучшую в классе производительность и возможности совместной работы Office 365 с решениями для управления устройствами и обеспечения безопасности для защиты бизнес-данных. Клиент Microsoft 365 бизнес готов перенести свою ИТ-инфраструктуру в облако и заинтересован в упреждающем подходе, чтобы защитить данные как на корпоративных устройствах, так и на устройствах сотрудников.

### <a name="how-can-i-get-microsoft-365-business-for-my-business"></a>Как получить Microsoft 365 бизнес для моей организации? 
Microsoft 365 бизнес можно приобрести через <a href="https://www.microsoft.com/solution-providers/search" target="_blank">партнера Майкрософт</a> или напрямую у <a href="https://www.microsoft.com//microsoft-365/business" target="_blank">Майкрософт</a>. При принятии решения о покупке непосредственно у корпорации Майкрософт или партнера Майкрософт, рекомендуется оценить возможности персонала и желание поддерживать ИТ-инфраструктуру. Партнер Майкрософт может помочь в развертывании ИТ-инфраструктуры и управлении ею, включая решения Майкрософт.

### <a name="how-much-does-microsoft-365-business-cost"></a>Сколько стоит Microsoft 365 бизнес? 
Microsoft 365 бизнес предлагается по цене 20,00 долларов США за пользователя в месяц при приобретении _годового контракта_ непосредственно у Майкрософт. При покупке через партнеров Майкрософт цены могут меняться в зависимости от предоставляемых партнером услуг и его модели ценообразования для Microsoft 365 бизнес. 

### <a name="is-there-a-cap-to-how-many-microsoft-365-business-seats-a-customer-can-have"></a>Есть ли ограничение на количество рабочих мест Microsoft 365 бизнес, которое может приобрести клиент? 
Решение Microsoft 365 бизнес было разработана для малых и средних предприятий с низкими или средними требования к сложности ИТ-инфраструктуры. Клиенты могут приобрести до 300 лицензии на Microsoft 365 бизнес для своей организации. Клиенты могут комбинировать облачные подписки. Таким образом, в зависимости от ИТ-потребностей организации, клиенты могут добавить лицензии Microsoft 365 корпоративный в ту же учетную запись. 
 
При рассмотрении среды, включающей несколько типов подписки, клиентам следует обратиться к своим доверенным ИТ-консультантам, чтобы определить лучший способ защиты различных подписок и управления ими, так как Microsoft 365 бизнеса и Microsoft 365 корпоративный используют различные возможности для обеспечения безопасности приложений и данных и управления ими.

### <a name="can-i-combine-microsoft-365-business-with-other-microsoft-subscription-offerings"></a>Можно ли объединить Microsoft 365 бизнес с другими предложениями по подписки Майкрософт? 
Да, клиенты могут объединить свои подписки на Microsoft 365 бизнес с планами и надстройками Azure, Dynamics 365, Enterprise Mobility + Security и Office 365.

### <a name="is-everyone-in-my-business-required-to-have-a-microsoft-365-business-subscription"></a>Всем ли сотрудникам моей организации необходимо иметь подписку на Microsoft 365 бизнес? 
Нет, подписка на Microsoft 365 бизнес необходима не всем, однако преимущества управления и безопасности доступны только пользователям с устройствами, управляемыми с помощью подписки на Microsoft 365 бизнес. 
 
Стандартизация ИТ-среды необходима для сокращения затрат на обслуживание и обеспечение безопасности в будущем. Этого должны стремиться достичь все организации. Тем не менее мы понимаем, что некоторые клиенты в малых и средних компаниях зачастую обновляют программное обеспечение одновременно с обновлением оборудования, что занимает продолжительное время. Компании могут развернуть Microsoft 365 бизнес для части своей организации, но для обеспечения лучшей защиты конфиденциальных коммерческих данных и согласованности процессов совместной работы рекомендуется развертывать решение для всех пользователей.

### <a name="how-can-i-know-if-the-hardware-and-software-i-run-today-is-compatible-with-microsoft-365-business"></a>Как понять, совместимо ли используемое мной в данный момент оборудование и программное обеспечение сегодня с Microsoft 365 бизнес? 
Если на вашем устройстве установлена операционная система Windows 7 Профессиональная или более поздней версии, она, скорее всего, соответствует минимальным требованиям для Microsoft 365 Business. Для некоторых функций Windows 10, таких как Кортана, Windows Hello и мультисенсорных, требуется определенное оборудование, которое доступно только на новых компьютерах. Дополнительные сведения приведены в разделе <a href="https://www.microsoft.com/windows/windows-10-specifications" target="_blank">требования к системе для Windows 10 Pro</a> . 
 
Ожидается, что большинство существующих классических приложений (Win32) будут поддерживаться в Windows 10 без каких-либо изменений. Клиентам и их доверенным ИТ-консультантам следует ознакомиться с рекомендуемым процессом тестирования приложений на <a href="https://docs.microsoft.com/windows/deployment/planning/windows-10-compatibility#recommended-application-testing-process" target="_blank">совместимость с Windows 10</a> и <a href="https://products.office.com/office-system-requirements#subscription-plans-section" target="_blank">системными требованиями для Office</a> для упрощения перехода на Microsoft 365 бизнес. 

### <a name="what-is-windows-10-business"></a>Что такое Windows 10 для бизнеса? 
Windows 10 бизнеса — это набор облачных служб и возможностей управления устройствами, которые дополняют Windows 10 Pro и обеспечивают централизованное управление и контроль безопасности Microsoft 365 бизнес. Windows 10 для бизнеса также поставляется вместе со службой Windows AutoPilot, которая упрощает развертывание новых компьютеров с Windows 10. Если у вас есть устройства с лицензией на Windows 7, 8 и 8.1 Профессиональная, Microsoft 365 бизнес, предоставляет обновление до Windows 10 Pro, которое является необходимым условием для развертывания Windows 10 для бизнеса. 

### <a name="how-does-microsoft-365-business-help-support-our-bring-your-own-device-byod-policy"></a>Как Microsoft 365 бизнес обеспечивает поддержку политики использования собственных устройств (BYOD) в моей компании? 
Многие сотрудники предпочитают использовать свои собственные мобильные телефоны или планшеты для доступа к личной и рабочей информации вместо использования нескольких устройств для различных целей. Использование личных устройств для работы достаточно распространено, но повышает риск того, что коммерческая информация окажется в чужих руках. Многие конкурирующие мобильной решения для защиты данных требуют переключения устройства в определенный режим или использования другого сложного механизма, которого пользователя будут избегать по причине его сложности. 
 
Microsoft 365 Business предлагает пользователям простое, но мощное средство, позволяющее сотрудникам использовать персональные устройства для работы, в то же время предоставляя возможность предотвратить доступ к бизнес-информации для этих устройств, их хранение и/или совместное использование. . Это обеспечивается следующими средствами. 
* **Защита мобильных приложений Office** помогает приложениям защитить данные Office, включая электронную почту, календарь, контакты и документы на мобильных устройствах под управлением iOS и Android путем применения политик, таких как автоматическое удаление коммерческих данных спустя определенный промежуток времени, в течение которого не выполняется подключение к службе, хранение данных только в OneDrive для бизнеса, обязательная проверка ПИН-кода или отпечатка пальца для получения доступа к приложениям Office и предотвращение копирования данных компании из приложений Office в личные приложения. Управление мобильными приложениями для других мобильных приложений с помощью Intune также доступно для подписчиков Microsoft 365 Business. 
  
* **Управление устройствами для компьютеров с Windows 10** позволяет компаниям самостоятельно устанавливать и поддерживать такие возможности, как защита от вредоносных программ с помощью Защитника Windows, автоматические обновления и отключение экрана по истечении предписанного промежутка времени. Кроме того, с потерянных или украденных устройств с Windows 10 можно полностью удалить бизнес-приложения и данные с помощью центра администрирования. 

* **Управление устройствами для iOS, Android _Амп_ MacOS** Features помогает предприятиям безопасно управлять разнообразной экосистемой устройств, включающей устройства iOS, Android, Windows и MacOS. 

### <a name="how-does-microsoft-365-business-help-protect-pcs-in-my-organization-from-malicious-attacks"></a>Как Microsoft 365 бизнеса помогает защитить компьютеры в организации от вредоносных атак? 
ПК под управлением Microsoft 365 Business защищены защитником Windows, который является недоступным. 1 антивирусная функция в Windows 10 защищает больше компьютеров от вирусов, вредоносных программ, шпионских программ и других угроз, чем любое другое решение.  С помощью Microsoft 365 Business предприятия могут обеспечить работу системы защиты защитника Windows и всегда актуальны все свои устройства с Windows 10.  
 
Windows 10 бизнес также включает в себя защитник Windows Defender, новый набор возможностей предотвращения вторжения. Один из его компонентов, контролируемого доступа к папкам, перестает отвечать на записи, блокируя папки и предотвращая доступ несанкционированных приложений к важным файлам пользователя. 

### <a name="whats-the-difference-between-office-365-business-premium-microsoft-365-business-and-microsoft-365-enterprise"></a>В чем разница между Office 365 бизнес премиум, Microsoft 365 бизнес и Microsoft 365 корпоративный? 
Корпорация Майкрософт предоставляет разнообразные решения для обеспечения производительности и безопасности, которые компании малого и среднего бизнеса могут выбрать при обновлении инфраструктуры рабочих столов и устройств. Некоторые из этих решений предоставляют более мощные возможности, чем другие. 

**Office 365 бизнес премиум** обеспечивает лучшую в классе производительность благодаря приложениям и службам Office 365, но не включает возможности защиты приложений и управления устройствами, доступные в Microsoft 365 бизнес.

**Microsoft 365 бизнес** сочетает в себе приложения и службы Office 365, службы управления мобильными приложениями и Windows 10 Pro, которая обеспечивает удаленное управление устройствами и помогает защитить их от вирусов и вредоносных программ. Этот выпуск включает упрощенную консоль управления, посредством которой осуществляется администрирование политик устройств и данных. Большинство компаний малого и среднего размера можно использовать в Microsoft 365 Business. 
 
**Microsoft 365 корпоративный** это набор планов лицензирования, которые обеспечивают повышенный уровень соответствия требованиям и управления безопасностью в Microsoft 365 бизнес и предназначены для корпоративных клиентов и клиентов с более чем 300 пользователей. Кроме того, в планах Microsoft 365 Enterprise предусмотрены дополнительные функции, в том числе средства бизнес-аналитики и аналитики.

### <a name="can-i-switch-my-office-365-plan-to-microsoft-365-business"></a>Можно ли перейти с плана Office 365 на Microsoft 365 бизнес? 
Да, пользователи могут изменить свой план с соответствующего условиям плана Office 365 на Microsoft 365 бизнес. В зависимости от текущего плана клиента возможно уменьшение или увеличение ежемесячной платы.

### <a name="in-what-regions-is-microsoft-365-business-available"></a>В каких регионах доступен Microsoft 365 бизнес? 
Microsoft 365 бизнес будет доступно для всех партнеров и клиентов там, где доступна подписка на Office 365. См. список <a href="https://products.office.com/business/international-availability" target="_blank">доступности Office 365 для языков, стран и регионов</a>. 

### <a name="is-there-a-microsoft-365-business-trial-i-may-use-to-evaluate-the-offer"></a>Существует ли пробная версия Microsoft 365 бизнес для оценки этого решения? 
Пробная версия Microsoft 365 Business доступна для поставщиков служб шифрования. Пробная версия для прямых клиентов будет доступна позже. 
 
### <a name="what-should-customers-and-partners-know-before-running-microsoft-365-business-within-their-organization"></a>Что клиентам и партнерам следует знать перед запуском Microsoft 365 бизнес в организации? 
Клиенты, которым необходимы все возможности Microsoft 365 бизнес, должны использовать на существующих компьютерах с Windows 7, 8.1 или 10 Pro<sup>[2](#footnote2)</sup>. На существующих компьютерах с Windows 10 Pro должно быть установлено обновление Creators Update. 

## <a name="deployment"></a>Развертывание

### <a name="what-should-customers-consider-when-planning-a-microsoft-365-business-deployment"></a>Что следует принять во внимание клиентам при планировании развертывания Microsoft 365 бизнес? 
Самый прямой путь к успешному развертыванию Microsoft 365 бизнес — это взаимодействие с партнером корпорации Майкрософт. Партнеры прошли обширную подготовку и обладают большим опытом работы с широким спектром пользовательских сценариев и лучше всего понимают ваши потребности и особенности вашей среды. Клиенты, имеющие в штате опытных ИТ-сотрудников, могут использовать раздел <a href="https://support.office.com/article/496e690b-b75d-4ff5-bf34-cc32905d0364" target="_blank">Начало работы с Microsoft 365 бизнес</a> для помощи в развертывания Microsoft 365 бизнес. 

### <a name="does-microsoft-365-business-include-the-full-capabilities-of-microsoft-intune"></a>Включает ли Microsoft 365 бизнес все возможности Microsoft Intune? 
Да, Microsoft 365 Business подписчики лицензированы для использования полных возможностей Intune для iOS, Android, MacOS и другого управления устройствами на разных платформах. Функции, недоступные в упрощенной консоли управления в Microsoft 365 бизнес, такие как управление сторонними приложениями и настройка профилей Wi-Fi, VPN-сертификаты и т. д., можно управлять в полной консоли Intune. У вас есть права на использование диспетчера конфигураций для управления клиентами, охваченными лицензией EMS. Запросы, связанные с SCCM, можно найти в статье <a href="https://docs.microsoft.com/en-us/sccm/core/understand/product-and-licensing-faq#bkmk_csp" target="_blank">SCCM FAQs</a>
  

### <a name="does-azure-active-directory-premium-p1-come-with-microsoft-365-business"></a>Поступили ли Azure Active Directory Premium P1 с Microsoft 365 Бизнес? 
В основе Microsoft 365 бизнес лежат различные технологии Майкрософт, и хотя это решение предоставляет некоторые возможности Azure Active Directory, оно не является полной версией. Политики безопасности и управления, созданные в Microsoft 365 Business, основываются на некоторых функциях Azure, но не включают все функции (например, функции самообслуживания, функции условного доступа и отчеты). Клиенты могут приобрести Azure Active Directory Premium P1 или P2 как надстройку Microsoft 365 бизнес. Дополнительные сведения см. в [описании Microsoft 365 Business Service](https://technet.microsoft.com/library/b042d1fd-c997-46d8-8c02-1df4c932f148) 
 
 
### <a name="does-microsoft-365-business-allow-customers-to-manage-macs"></a>Позволяет ли Microsoft 365 бизнес управлять компьютерами Mac? 
Intune помогает безопасно управлять устройствами iOS, Android, Windows и macOS. 
 
### <a name="what-is-windows-autopilot"></a>Что такое Windows AutoPilot? 
Windows AutoPilot — это служба, которая упрощает развертывание новых компьютеров с Windows 10. Этот процесс может быть выполнен, когда конечный пользователь входит в Microsoft 365 бизнеса в первый раз, за счет использования элементов централизованного управления Microsoft 365 бизнес без участия ИТ-отдела. Можно также использовать Windows AutoPilot для имеющихся компьютеров с Windows 10 Профессиональная Creators Update (или более поздней версией), для которых был выполнено восстановление до заводских настроек. Подробные сведения об автопилоте Windows можно найти в <a href="https://blogs.technet.microsoft.com/windowsitpro/2017/06/29/modernizing-windows-deployment-with-windows-autopilot/" target="_blank">этой публикации в июне, 2017 в блоге</a>.

## <a name="compatibility"></a>Совместимость

### <a name="can-i-add-office-365-add-ons-to-microsoft-365-business"></a>Можно ли добавить надстройки Office 365 в Microsoft 365 бизнес? 
Все надстройки, которые могут быть добавлены в Office 365 бизнес премиум, можно добавить в Microsoft 365 бизнес. Это означает, что вы можете приобрести Advanced Threat Protection, Office 365 Cloud App Security, Advanced Compliance, Threat Intelligence, MyAnalytics, PowerBI Pro и аудиоконференцию. 
 
### <a name="can-i-add-phone-system-and-calling-plans-to-microsoft-365-business"></a>Можно ли добавить планы "Телефонная система" и "Вызовы" в Microsoft 365 бизнес? 
Нет, эти планы зарезервированы для пользователей с более сложными потребностями. Клиентам, которым необходимы данные возможности, рекомендуется ознакомиться с предложениями Microsoft 365 корпоративный. 
 
### <a name="can-microsoft-365-business-customers-use-windows-defender-advanced-threat-protection"></a>Могут ли клиенты Microsoft 365 бизнес использовать службу Advanced Threat Protection в Защитнике Windows? 
Нет, пользователям, которым необходима служба Advanced Threat Protection в Защитнике Windows, потребуется Windows 10 Корпоративная E5 или Microsoft 365 корпоративный E5. 
 


### <a name="can-customers-create-hosted-windows-10-vms-with-a-microsoft-365-business-subscription"></a>Могут ли клиенты создавать размещенные виртуальные машины с Windows 10 с помощью подписки Microsoft 365 бизнес? 
Нет, пользователям, которым необходима виртуализация, следует приобрести Windows 10 Корпоративная или подписку на Microsoft 365 корпоративный.

### <a name="where-can-i-learn-more-about-the-opportunities-and-benefits-in-becoming-a-microsoft-partner"></a>Где можно узнать подробнее о возможностях и преимуществах получения статуса партнера Майкрософт?

### <a name="where-can-i-learn-how-to-sell-microsoft-365-business"></a>Где можно узнать, как продавать Microsoft 365 бизнес? 
Партнеры, которые в данный момент занимаются продажами Office 365, могут использовать те же методы консультационных продаж для продажи Microsoft 365 бизнес. Кроме того, мы представляем дополнительные ресурсы и обучение для специалистов по продажам, что помогает им понять потребности существующей среды клиентов, зависимость от Active Directory и их требования к мобильности и безопасности, чтобы эффективно рассказать обо всех преимуществах Microsoft 365 бизнес для определенного клиента. Эти ресурсы можно найти на портале Office для партнеров по адресу [http://partners.office.com/microsoft365business](http://partners.office.com/microsoft365business). 

### <a name="how-can-microsoft-365-business-help-partners-increase-the-profitability"></a>Как Microsoft 365 бизнес может помочь партнерам повысить рентабельность? 
Microsoft 365 бизнес поможет партнерам снизить затраты путем повышения операционной эффективности и выручки за счет продажи дополнительных услуг. Исследование Forrester Research "Общий экономический эффект Microsoft 365 бизнес", проведенное в июне 2017 года (https://partners.office.com/TEIBusiness), показывает, что Microsoft 365 бизнес положительно скажется на прибыльности партнеров. 

В рамках исследования партнеры сообщили, что благодаря Microsoft 365 бизнеса они достичь следующих показателей. 

- Увеличение выручки за \[разовые\] услуги развертывания и консультационные услуги на 20 процентных пунктов
- Увеличение коэффициента сопутствующих продаж управляемых услуг на 10 процентных пунктов
- Увеличение рентабельности консультационных услуг и \[регулярных\] услуг управления на 8 процентных пунктов (за счет снижения расходов)

### <a name="what-resources-are-available-to-partners-to-sell-deploy-and-support-microsoft-365-business"></a>Какие ресурсы доступны для партнерам для продажи, развертывания и поддержки Microsoft 365 бизнес?
Корпорация Майкрософт предоставляет широкий выбор ресурсов для партнеров CSP для продажи и поддержки Microsoft 365 бизнес. Их можно найти на странице [https://partners.office.com/microsoft365business](https://partners.office.com/microsoft365business).

### <a name="what-up-sell-opportunities-does-microsoft-365-business-give-partners"></a>Какие возможности для увеличения объема продаж предоставляет партнерам Microsoft 365 бизнес? 
Microsoft 365 бизнес позволяет партнерам поддерживать свое положение доверенных консультантов путем создания надежной и безопасной платформы, на основе которой можно продавать дополнительные услуги и обновлять существующие продукты и услуги. Microsoft 365 бизнес дает возможность обсудить обновление существующих продуктов с клиентами, использующих Exchange Server, Exchange Online или Office 365 бизнес базовый. Партнеры также могут дополнительно увеличить выручку за счет повышения объема управляемых услуг и стоимости поддержки каждого пользователя. 
 
С помощью нового компонента Windows AutoPilot, входящего в состав Microsoft 365 бизнес, партнеры, которые ранее не хотели продавать новые устройства с Windows из-за сложности логистики и расходов, сочтут эту возможность гораздо более привлекательной. Пользователи, которые уверены в безопасности своих локальных и мобильных устройств, с большей вероятностью приобретают дополнительные службы, такие как Dynamics 365. 
 
### <a name="should-partners-sell-microsoft-365-business-over-other-plans-from-microsoft"></a>Следует ли партнерам продавать Microsoft 365 бизнес через другие планы корпорации Майкрософт? 
Поставщику облачных решений Майкрософт всегда следует продавать тот план, который оптимально соответствует потребностям и бюджету клиента. Например, если клиент должен соблюдать нормативные требования безопасности и конфиденциальности, поставщики облачных решений могут продавать Microsoft 365 бизнес, а также надстройки, которые помогают клиенту соблюсти эти требования или способны предложить повышенную безопасность и возможности управления, предоставляемые выпусками Microsoft 365 бизнес E. 
 
### <a name="some-of-my-customers-have-devices-that-are-not-genuine-will-microsoft-365-business-make-these-devices-genuine"></a>У некоторых моих клиентов есть не подлинные устройства. Станут ли они подлинными после приобретения Microsoft 365 бизнес? 
Microsoft 365 бизнес не делает подлинными не подлинные версии Windows. Microsoft 365 для бизнеса предоставляет преимущества обновления, позволяя пользователям подлинных Windows 7, 8 или 8.1 Pro обновить их до самой последней подлинной версии Windows 10 Pro. 
 
### <a name="what-support-is-available-to-csp-partners-for-the-microsoft-365-business-preview"></a>Какая поддержка доступна партнерам CSP для предварительной версии Microsoft 365 бизнес? 
Те же каналы поддержки, которые доступны партнерам CSP сейчас (поддержка Premier и программа расширенной поддержки), подготовлены для поддержки Microsoft 365 бизнес и готовы предоставлять ее партнерам. 

### <a name="what-is-the-gdpr-and-how-does-microsoft-365-business-help-customers-with-their-compliance-obligations"></a>Что такое GDPR и как Microsoft 365 бизнес помогает клиентам выполнять свои обязательства по соблюдению нормативных требований?
Общие правила защиты данных (GDPR) — это совершенно новый регламент о конфиденциальности, который предоставляет гражданам Европейского Союза (ЕС) возможность лучше контролировать свои персональные данные, а также требует от организаций обеспечить защиту этих персональных данных. GDPR требует от организаций, контролирующих персональные данные граждан ЕС или управляющих ими, использования только тех сторонних обработчиков данных, которые отвечают требованиям GDPR в отношении обработки персональных данных. В марте 2017 года корпорация Майкрософт предоставила возможность следующих договорных гарантий. Пользователи, у которых есть вопросы относительно того, как корпорация Майкрософт может помочь им удовлетворить свои дополнительные обязательства по GDPR, должны узнать о расширенных функциях обеспечения соответствия требованиям и безопасности, доступных в виде надстроек (например, Azure Information Protection, защита от потери данных, расширенный Защита от угроз и т. д.) и в других наборах (например, Microsoft 365 Enterprise ~). Для получения дополнительных сведений посетите страницу [www.microsoft.com/gdpr](https://www.microsoft.com/gdpr).
