# <a name="transition-a-microsoft-365-business-csp-subscription"></a>Перевод подписки CSP Microsoft 365 бизнес

Если у вас есть подписка CSP на предварительную версию Microsoft 365 бизнес, то в этом руководстве вы узнаете, как перевести существующую подписку с предварительной версии Microsoft 365 бизнес на общедоступную.

**Процедура перевода подписки с предварительной версии на общедоступную**

1. Войдите в <a href="https://partnercenter.microsoft.com" target="_blank">Центр партнеров</a>.
2. На информационной панели выберите **Клиенты**, а затем найдите и выберите название компании.

    Отобразится перечень подписок выбранной компании.

    ![Подписки клиента в Центре партнеров](images/pc_customer_subscriptions_1.png)
    
3. На странице **Подписки** этой компании выберите **Добавить подписку**.
4. На странице **Новая подписка** выберите **Малый бизнес**, а затем в списке выберите пункт **Microsoft 365 бизнес**.
5. Добавьте количество лицензий и выберите **Далее: просмотр**, чтобы просмотреть подписку, а затем выберите **Отправить**.

    ![Просмотр новой подписки на Microsoft 365 бизнес](images/pc_customer_reviewnewsubscription.png)

    В разделе **Подписки на основе лицензии** отобразятся пункты **Предварительная версия Microsoft 365 бизнес** и **Microsoft 365 бизнес**. Далее потребуется приостановить подписку на предварительную версию.

6. Выберите **Предварительная версия Microsoft 365 бизнес**.
7. На странице **Предварительная версия Microsoft 365 бизнес** выберите **Приостановлено**, чтобы приостановить действие подписки на предварительную версию.

    ![Приостановка подписки на предварительную версию Microsoft 365 бизнес](images/pc_customer_m365bpreview_suspend.png)

8. Выберите **Отправить** для подтверждения.

    На странице **Подписки** убедитесь, что для пункта **Предварительная версия Microsoft 365 бизнес** отображается состояние **Приостановлено**.

    ![Проверка приостановленного состояния подписки на предварительную версию](images/pc_customer_m365bpreview_suspend_confirm.png)

9. Кроме того, можно проверить лицензионное соглашение. Для этого выполните следующие действия:
    1. Выберите **Пользователи и лицензии** на странице **Подписки** компании.
    2. На странице **Пользователи и лицензии** выберите пользователя.
    3. На странице пользователя убедитесь, что в разделе **Назначение лицензий** отображается **Microsoft 365 бизнес**.

        ![Проверка назначения лицензии Microsoft 365 бизнес пользователю](images/pc_customer_userslicenses_m365b_validate.png)

## <a name="impact-to-customers-and-users-during-and-after-transition"></a>Влияние на клиентов и пользователей во время и после перехода

Во время и после перехода клиенты и пользователи никак не затрагиваются.

## <a name="mobile-device-impacts-upon-subscription-expiration"></a>Влияние истечения срока действия подписки на мобильные устройства

В следующей таблице представлена сводная информация о влиянии на политики управления приложениями на мобильных устройствах.

|                            | Полная лицензия                      | 60 дней после истечения срока действия подписки          |
|----------------------------|------------------------------------------------|------------------------------------|
| **Удаление рабочих файлов с неактивного устройства** | Рабочие файлы удаляются по прошествии указанного количества дней | Рабочие файлы остаются на личных устройствах пользователей |
| **Принудительное сохранение рабочих файлов пользователей в OneDrive для бизнеса** | Рабочие файлы можно сохранить только в OneDrive для бизнеса | Рабочие файлы можно сохранить в любом месте |
| **Шифрование рабочих файлов** | Рабочие файлы шифруются | Рабочие файлы больше не шифруются.</br> Удаляются политики безопасности и данные Office в приложениях. |
| **Для доступа к приложениям Office требуется PIN-код или отпечаток пальца** | Ограниченный доступ к приложениям | Нет ограничений доступа на уровне приложений |
| **Сброс PIN-кода при ошибке входа** | Ограниченный доступ к приложениям | Нет ограничений доступа на уровне приложений |
| **После простоя приложений Office пользователям требуется повторно выполнить вход** | Требуется вход | Для доступа к приложениям вход не требуется |
| **Запрет доступа к рабочим файлам на устройствах со снятой защитой или с административным доступом** | Доступ к рабочим файлам на устройствах со снятой защитой или с административным доступом запрещен | Доступ к рабочим файлам на устройствах со снятой защитой или с административным доступом разрешен |
| **Разрешение пользователям копировать содержимое из приложений Office в личные приложения** | Копирование и вставка ограничены приложениями, доступными в составе подписки на Microsoft 365 бизнес | Копирование и вставка доступны во всех приложениях |
