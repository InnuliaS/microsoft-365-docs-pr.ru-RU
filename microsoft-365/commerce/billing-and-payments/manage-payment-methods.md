# <a name="manage-payment-methods"></a>Управление методами оплаты

При покупке бизнес-продуктов или услуг корпорации Майкрософт вы можете использовать существующий метод оплаты или добавить новый. Вы можете использовать кредитную или дебетовую карту или банковский счет для оплаты приобретенных вами вещей. Но вы можете управлять только методами оплаты, которые вы добавляете.

> [!NOTE]
> Возможность оплаты с помощью банковского счета недоступна в некоторых странах или регионах.
>
> Необходимо использовать метод оплаты, выданный из той же страны, что и клиент.

## <a name="add-a-payment-method"></a>Добавление способа оплаты

Добавление метода оплаты не связывает с ним какие бы то ни было подписки. Чтобы назначить одну подписку методу оплаты, ознакомьтесь со статьей [изменение способа оплаты для одной подписки](#change-a-payment-method-for-a-single-subscription). Чтобы заменить все подписки, использующие другой способ оплаты, на новый, ознакомьтесь со статьей [Замена метода оплаты](#replace-a-payment-method).

1. В центре администрирования перейдите на > страницу<a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">платежных</a> **ведомостей &** > оплаты оплаты оплате **оплате**.
2. Выберите элемент **Добавление метода оплаты**.
3. На странице **Методы оплаты** выберите метод оплаты в раскрывающемся меню.
4. Введите сведения о новой карточке или банковском счете, а затем нажмите кнопку **Добавить**.

## <a name="update-payment-method-details"></a>Обновление сведений о методе оплаты

Вы можете изменить имя для кредитной или дебетовой карты, адреса выставления счетов или даты истечения срока действия для существующего метода оплаты. Однако вы не можете изменить карточку или номер учетной записи. Если номер учетной записи изменился, [замените его на другой способ оплаты](#replace-a-payment-method), а затем [удалите старый](#delete-a-payment-method).

1. В центре администрирования перейдите на > страницу<a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">платежных</a> **ведомостей &** > оплаты оплаты оплате **оплате**.
2. Выберите строку метода оплаты, который необходимо обновить. В правой области нажмите кнопку **изменить**.
3. Обновите сведения о методе оплаты (имя на кредитной или дебетовой карте, адресе выставления счета или дату окончания срока действия), а затем нажмите кнопку **сохранить**.

## <a name="replace-a-payment-method"></a>Замена способа оплаты

При замене способа оплаты вы заменили его для всех подписок и профилей выставления счетов, использующих один и тот же способ оплаты. При замене метода платежа существующий метод оплаты не удаляется. Он по-прежнему доступен для выбора и использования для других подписок и профилей выставления счетов.

Чтобы изменить способ оплаты для одной подписки, ознакомьтесь со статьей [изменение способа оплаты для одной подписки](#change-a-payment-method-for-a-single-subscription).

1. В центре администрирования перейдите на > страницу<a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">платежных</a> **ведомостей &** > оплаты оплаты оплате **оплате**.
2. Выберите строку метода оплаты для замены. В области справа перечислены все профили выставления счетов и отдельные подписки, использующие выбранный способ оплаты.
3. В области справа выберите **заменить метод оплаты для всех элементов**.
4. Чтобы использовать существующий метод оплаты, выберите его в раскрывающемся списке, а затем нажмите **заменить**.
    > [!NOTE]
    > Если у вас есть подписки, связанные с профилем выставления счетов, вы можете оплатить их только с помощью кредитных или дебетовых карт. Если у вас есть банковские счета, указанные на странице **способы оплаты** , они недоступны для выбора в раскрывающемся списке.
5. Чтобы добавить новый метод оплаты, нажмите кнопку **Добавить метод оплаты**.
6. В области **метод добавления оплаты** введите сведения об учетной записи, а затем нажмите кнопку **сохранить**. Необходимо использовать метод оплаты из той же страны, что и клиент.
7. Новый метод оплаты уже выбран в раскрывающемся списке. Нажмите кнопку **Replace** (Заменить).

## <a name="change-a-payment-method-for-a-single-subscription"></a>Изменение способа оплаты для одной подписки

Вы можете изменить способ оплаты, используемый для оплаты одной подписки.

1. В центре администрирования перейдите на страницу **выставление счетов** > по<a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">продуктам</a> .
2. На вкладке **подписки** выберите подписку, с которой вы хотите платить, с помощью альтернативного метода оплаты. 
3. В разделе **выставление счетов**рядом с пунктом метод оплаты нажмите кнопку **изменить**.
4. Рядом с существующим методом оплаты нажмите кнопку **изменить**.
5. В раскрывающемся списке выберите альтернативный способ оплаты или нажмите, чтобы добавить метод оплаты.
6. Если вы добавляете метод оплаты, введите сведения об карточке или учетной записи, а затем нажмите кнопку **сохранить**.
7. Убедитесь, что выбранный метод оплаты задан правильно, а затем нажмите кнопку **сохранить**.

## <a name="delete-a-payment-method"></a>Удаление метода оплаты

Можно удалить только метод оплаты, не присоединенный к подписке или профилю выставления счетов. Это относится ко всем подпискам, независимо от их состояния.

### <a name="delete-a-payment-method-with-no-subscriptions-or-billing-profiles-attached"></a>Удаление метода оплаты без подсоединенных подписок или профилей выставления счетов

Если способ оплаты не связан с подписками или профилями выставления счетов, можно сразу удалить его.

1. В центре администрирования перейдите на > страницу<a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">платежных</a> **ведомостей &** > оплаты оплаты оплате **оплате**.
2. Найдите метод оплаты, который необходимо удалить, выберите три точки, а затем нажмите кнопку **Удалить**.
3. В нижней части правой области нажмите кнопку **Удалить**.

### <a name="delete-a-payment-method-with-subscriptions-or-billing-profiles-attached"></a>Удаление метода оплаты с присоединенными подписками или профилями выставления счетов

Если метод оплаты присоединен к любой подписку или к профилям выставления счетов, сначала замените его на существующий метод оплаты или добавьте новый, а затем удалите старый метод оплаты.

1. В центре администрирования перейдите на > страницу<a href="https://go.microsoft.com/fwlink/p/?linkid=2018806" target="_blank">платежных</a> **ведомостей &** > оплаты оплаты оплате **оплате**.
2. Выберите строку для удаляемого метода оплаты. В правой области перечислены существующие подписки, которые используют этот способ оплаты.
3. В правой области нажмите кнопку **Удалить**.
4. Чтобы использовать существующий метод оплаты, выберите его в раскрывающемся списке, нажмите кнопку **Далее**, а затем выберите **Удалить**.
    > [!NOTE]
    > Если у вас есть подписки, связанные с профилем выставления счетов, вы можете оплатить их только с помощью кредитной карты. Если у вас есть банковские счета, указанные на странице **способы оплаты** , они недоступны для выбора в раскрывающемся списке.
5. Чтобы добавить новый метод оплаты, нажмите кнопку **Добавить метод оплаты**.
6. Выберите тип способа оплаты, который требуется добавить, введите сведения об учетной записи, а затем нажмите кнопку **сохранить**.
7. Новый метод оплаты уже выбран в раскрывающемся списке. Нажмите кнопку **Далее**.
8. Выберите **Удалить**.
