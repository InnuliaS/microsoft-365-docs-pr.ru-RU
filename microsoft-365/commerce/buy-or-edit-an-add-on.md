# <a name="buy-or-edit-an-add-on-for-office-365-for-business"></a>Покупка и изменение надстроек для Office 365 бизнес

Для некоторых планов Office 365 для бизнеса можно приобрести дополнительные надстройки, чтобы расширить возможности своей подписки.


::: moniker range="o365-worldwide"

> [!NOTE]
> Если вы не используете новый Центр администрирования Microsoft 365, можно включить его с помощью переключателя **Попробовать новый Центр администрирования**, расположенного в верхней части главной страницы.

::: moniker-end 

## <a name="buy-an-add-on"></a>Приобретение надстройки

::: moniker range="o365-worldwide"

1. В центре администрирования перейдите на страницу Услуги по **выставлению счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=868433" target="_blank">покупки</a> .

2. В нижней части страницы **услуги покупки** выберите **надстройки**.

3. На странице " **Покупка служб** " выберите подписку на надстройку, которую вы хотите купить.

4. Введите необходимое количество лицензий и укажите, следует ли платить ежемесячно или за весь год. Укажите, хотите ли вы автоматически назначать лицензии всем пользователям, у которых в настоящее время нет лицензии. Если отображается, выберите базовую подписку.

5. Нажмите **кнопку извлечь.**

6. Ознакомьтесь со сведениями о ценах, а затем нажмите кнопку **Далее**.

7. Введите сведения об оплате, а затем выберите пункт **разместить заказ** \> **перейдите в домашнюю страницу администратора**.

::: moniker-end

## <a name="update-an-add-on"></a>Обновление надстройки

Действия, которые необходимо выполнить для обновления надстройки, зависят от того, является ли надстройка традиционной или отдельной надстройкой. 

*Традиционные надстройки* связаны с определенной подпиской. При отмене подписки связанная надстройка также отменяется.
  
*Изолированные надстройки* не связаны с определенной подпиской. Изолированные надстройки отображаются как отдельные подписки на странице " **подписки** " (или на странице " **продукты & Services** " в новом центре администрирования) и имеют свой срок действия. Управлять изолированной надстройкой можно так же, как и при управлении любой другой подпиской.
  
### <a name="update-a-traditional-add-on"></a>Обновление традиционной надстройки

::: moniker range="o365-worldwide"

1. В Центре администрирования перейдите на страницу **Выставление счетов** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">Продукты и службы</a>.

2. На странице " **продукты & службы** " выберите подписку, для которой вы хотите обновить надстройку.

3. Выберите **надстройки**. Будут показаны приобретенные надстройки.

4. Выберите **изменить количество** для надстройки, которую вы хотите изменить.

5. Введите требуемое количество изменений для подписки, а затем нажмите кнопку **послать изменение**.

::: moniker-end


::: moniker range="o365-germany"

1. В центре администрирования перейдите на страницу <a href="https://go.microsoft.com/fwlink/p/?linkid=847745" target="_blank">подписки</a> на **выставление счетов** \> .

2. На странице " **подписки** " выберите подписку, для которой вы хотите обновить надстройку.

3. Выберите **надстройки**.

    Отобразятся приобретенные вами надстройки, для каждой из которых будет доступна ссылка **Изменить количество**.

4. Выберите ссылку **изменить количество** для надстройки, которую вы хотите обновить. 

    ![Окно дополнительных компонентов со ссылкой на изменение количества.](../admin/media/fa7c39f4-5256-49bb-9a26-70f1a52ae926.png)
  
5. Введите требуемое количество пользовательских лицензий и нажмите кнопку " **послать**".

    > [!TIP]
    > Вы также можете изменить количество пользовательских лицензий с помощью стрелок или путем ввода нужного числа в соответствующее поле.
  
    ![Поле "Всего лицензий" и кнопка "Отправить" в диалоговом окне "Добавление и удаление лицензий".](../admin/media/dfee31e0-092d-4565-9bf5-ff3909b771e1.png)

## <a name="remove-an-add-on"></a>Удаление надстройки

Вы можете отменить отдельную надстройку так же, как вы [отмените подписку](subscriptions/cancel-your-subscription.md). Вы также можете настроить **периодическое выставление счетов** для автономной надстройки, выполнив описанные выше действия. Тем не менее традиционная надстройка не может быть удалена после ее приобретения. Если вам нужно удалить традиционную надстройку, [обратитесь за помощью в службу поддержки](../admin/contact-support-for-business-products.md).
