# <a name="document-metadata-fields-in-advanced-ediscovery"></a>Поля метаданных документа в Advanced eDiscovery

В приведенной ниже таблице перечислены поля метаданных для документов в наборе рецензий в случае расширенного обнаружения электронных данных. В таблице представлены следующие сведения:

- Имя поля метаданных (в столбце **имя поля** ).

- Имя свойства, которое можно искать при выполнении [запроса на проверку набора](review-set-search.md) (в столбце **имя поля для поиска** ). Пустая ячейка означает, что вы не можете выполнять поиск этого поля в запросе на проверку набора.

- Имя поля метаданных, которое включается при экспорте документов (в столбце **имя экспортируемого поля** ).  Пустая ячейка означает, что поле метаданных не включается в экспортированные метаданные.

- Имя поля метаданных, которое отображается при просмотре метаданных файла выбранного документа в наборе рецензирования (в столбце **имя отображаемого поля** ). Пустая ячейка означает, что поле Meta Data не включается при просмотре метаданных файла документа.

- Описание поля Meta Data (в **столбце Описание**).
