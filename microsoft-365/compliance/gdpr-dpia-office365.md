<span data-ttu-id="66932-p101">В разделе Общие данные защиты нормы (GDPR), контроллеры данных являются обязательными для подготовки оценки влияния данных защиты (DPIA) для обработки операций, которые являются «может привести к высокого риска для прав и freedoms физических лиц.» Нет ничего непосредственно в Microsoft Office 365, обязательно потребует создания DPIA контроллером данных с помощью его.
