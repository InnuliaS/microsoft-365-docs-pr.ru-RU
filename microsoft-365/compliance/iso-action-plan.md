<span data-ttu-id="df7bc-p101">Международная организация по стандартизации (ISO) является независимым негосударственным разработчиком рекомендательных международных стандартов. IEC возглавляет процесс подготовки и публикации международных стандартов для электрических, электронных и смежных технологий. В семействе стандартов 27000 ISO/IEC перечислены правила и механизмы, которые помогают поддерживать безопасность информационных ресурсов.</span><span class="sxs-lookup"><span data-stu-id="df7bc-p101">The International Organization for Standardization (ISO) is an independent nongovernmental developer of voluntary international standards. The ISO/IEC 27000 family of standards outlines controls and mechanisms that help maintain the security of information assets.</span></span>

<span data-ttu-id="df7bc-p102">ISO/IEC 27001 является международным стандартом по внедрению систем управления информационной безопасностью (ISMS). В ISMS описаны необходимые для использования методы и доказательства, связанные с требованиями, которые обязательны для надежного управления безопасностью информационных ресурсов в организации любого типа.</span><span class="sxs-lookup"><span data-stu-id="df7bc-p102">ISO/IEC 27001 is the international standard for implementing an information security management system (ISMS).
