# <a name="deploy-microsoft-365-enterprise"></a>Развертывание Microsoft 365 корпоративный

Microsoft 365 корпоративный — это сочетание продуктов Office 365, Enterprise Mobility + Security (EMS) и Windows 10 Корпоративная с перечисленными ниже функциями и возможностями. 

- Наличие интеллектуальной системы безопасности.
- Интеграция для простоты использования.
- Возможности для творчества.
- Поддержка работы в группах.

Чтобы использовать указанные выше возможности, недостаточно просто приобрести лицензии на эти три продукта. Необходимо развернуть продукты и их компоненты особым способом. В данной документации содержатся инструкции по развертыванию и правильной настройке этих продуктов и их компонентов.

При развертывании Microsoft 365 корпоративный используется два указанных ниже основных этапа.

1. Сначала необходимо развернуть обязательную [базовую инфраструктуру](deploy-foundation-infrastructure.md) со встроенными функциями обеспечения безопасности и интеграции, упрощающими управление, в частности обновление клиентского программного обеспечения и повышение производительности и уровня защиты.
2. Затем на основе базовой инфраструктуры необходимо развернуть набор необязательных [рабочих нагрузок и сценариев](deploy-workloads.md) для выполнения творческих и групповых задач в организации.

На рисунке ниже показана связь между базовой инфраструктурой и рабочими нагрузками и сценариями, а также порядок развертывания и настройки компонентов.

![](./media/deploy-microsoft-365-enterprise/m365-deploy-content-arch.png)

Средства обеспечения безопасности имеются на всех этапах базовой инфраструктуры.

## <a name="fasttrack"></a>FastTrack

FastTrack — это преимущество, которым можно воспользоваться в любой момент и не один раз. Это средство доступно в рамках вашей подписки. Инженеры Корпорации Майкрософт создали его, чтобы упростить вам переход в облако удобным для вас способом. Кроме того, с помощью FastTrack вы можете получить доступ к дополнительным услугам наших квалифицированных партнеров (при необходимости). На данный момент у нас более 40 000 клиентов, и FastTrack помогает достичь им максимальной рентабельности инвестиций, ускорить развертывание и повысить степень внедрения необходимых технологий в организации. См. статью [FastTrack для Microsoft 365](https://fasttrack.microsoft.com/microsoft365).

Воспользоваться преимуществами FastTrack для развертывания Microsoft 365 корпоративный позволит [помощник по развертыванию Microsoft 365](https://aka.ms/microsoft365setupguide) FastTrack. Он предоставляет рекомендации по развертыванию и настройке базовой инфраструктуры. Чтобы получить доступ к этой странице, необходимо выполнить вход от имени глобального администратора клиента Office 365 или Microsoft 365.

## <a name="take-a-test-drive"></a>Использование тестового выпуска

Поэкспериментируйте с Microsoft 365 корпоративный перед развертыванием в рабочей среде или развертыванием определенных конфигураций, используя руководства по лаборатории тестирования. Эти руководства представляют собой модульные учебники по настройке инфраструктуры или функции в упрощенной и наглядной среде для пробных или платных подписок. 

С помощью руководств по лаборатории тестирования вы можете продемонстрировать, настроить или создать экспериментальный вариант сложной конфигурации, рабочего процесса или полноценного сценария.

Дополнительные сведения см. в статье [Руководства по лаборатории тестирования для Microsoft 365 корпоративный](m365-enterprise-test-lab-guides.md).
