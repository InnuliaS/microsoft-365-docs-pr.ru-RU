<span data-ttu-id="ab003-p116">Параметры одноранговой передачи все чаще используются для миграции в Windows 10, создания образов компьютеров, обновлений программного обеспечения и персонализации пользователей. <span data-ttu-id="ab003-p117">**BranchCache** может помочь вам скачивать контент в распределенных средах, не перегружая сеть.
