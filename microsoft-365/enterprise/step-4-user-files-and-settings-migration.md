<span data-ttu-id="0189b-p109">USMT собирает учетные записи пользователей, их файлы, параметры операционной системы и настройки приложений, а затем переносит их в новый экземпляр Windows. <span data-ttu-id="0189b-p110">В ходе миграции вам потребуется достаточно места на сервере, чтобы использовать его в качестве временного хранилища для миграции.
