# <a name="azure-active-directory-tenant-details"></a>Сведения о клиенте Azure Active Directory
{Гори сведения об учетных записях, вызовах API, доступах и т. д.}


## <a name="data-transmitted-to-and-from-your-aad-account"></a>Данные, передаваемые в учетную запись AAD и получаемые от нее


Данные, отправляемые в учетную запись Майкрософт:

- Имена групп
- Настройка политики безопасности
- Заказы на устройства
- Учетные записи пользователей (мсадмин, MSTest, мваас_сок_ро, мваас_вдгсок)
- Назначение лицензии для учетных записей пользователей
- Политики обновления Windows для звонков обновления

Данные, отправляемые в корпорацию Майкрософт из вашей учетной записи:

- Данные об обновлениях, использовании и надежности устройств
- Данные о развертывании и надежности приложений
- Данные о развертывании политик обновления и безопасности
- Пользователи, назначенные устройствам  

Данные, передаваемые из учетной записи, хранятся в базах данных Azure SQL в клиенте Майкрософт, размещенном в США. Данные хранятся и обрабатываются в соответствии с политиками, описанными в статье {Microsoft Azure Security}. 

## <a name="api-permissions-and-calls"></a>Разрешения и вызовы API

**Во время регистрации:**

Разрешения API:
- DeviceManagementServiceConfig.ReadWrite.All
- Directory.AccessAsUser.All
- User.ReadWrite.All
- DeviceManagementConfiguration.ReadWrite.All
- DeviceManagementManagedDevices.ReadWrite.All
- Group.ReadWrite.All

Вызовов API:
- POST/Организатион/{организатионид}/сетмобиледевицеманажементаусорити
- GET/POST/Директориролес/{ИД}/мемберс
- GET/POST/Users
- GET/POST/граупс
- Исправление/граупс/{ИД}
- GET/POST/Девицеманажемент/девицеконфигуратионс
- ПОЛУЧЕНИЕ/Девицеманажемент/детектедаппс

**После регистрации в обычном управлении:**

Разрешения API:
- DeviceManagementManagedDevices.ReadWrite.All
- DeviceManagementApps.ReadWrite.All
- DeviceManagementConfiguration.ReadWrite.All
- Reports.Read.All
- User.ReadWrite.All
- Group.ReadWrite.All
- Directory.AccessAsUser.All

Вызовов API:
- GET/POST/Директориролес/{ИД}/мемберс
- GET/PATCH/POST/Users
- GET/POST/граупс
- Исправление/граупс/{ИД}
- GET/POST/Девицеманажемент/девицеконфигуратионс
- GET/POST/Девицеаппманажемент/мобилеаппс
- ПОЛУЧЕНИЕ/Девицеманажемент/детектедаппс
- ПОЛУЧЕНИЕ/девицес
- POST/усерс/{ИД | userPrincipalName}/Ассигнлиценсе
- ПОЛУЧЕНИЕ/Субскрибедскус
