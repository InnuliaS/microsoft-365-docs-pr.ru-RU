# <a name="admin-support-for-microsoft-managed-desktop"></a>Поддержка администраторов в отношении компьютеров, управляемых Майкрософт

Можно отправлять запросы свои отзывы и предложения в корпорацию Майкрософт, с помощью управляемого рабочего стола административные портал Microsoft или запросами в службу поддержки. Запросы на поддержку всегда приоритеты по отправке свои отзывы и предложения. Запросы на поддержку рассмотрены и управляются в соответствии с уровень серьезности, как описано в [таблице определения уровень серьезности](#sev). Отзывы рассматриваются и ответа предоставляются при запросе. 

>[!IMPORTANT]
>Убедитесь, что вы [set up для администрирования контакта](../get-started/add-admin-contacts.md) для упаковки приложения, устройства, безопасности и другие. Не удается отправить запрос поддержки в любом из этих областей, если не настроена для администрирования контакта.

Чтобы отправить запрос поддержки, откройте портал администрирования Microsoft Desktop управляемых и выберите **+ новый билет поддержки** на вкладке запрос на поддержку. Приведенные ниже сведения помогут guide варианта, в зависимости от используемого сценария. Когда вы выбираете уровень серьезности, ознакомьтесь с ожидания в **Ожидаемый ответ клиента**.

При создании запроса поддержки, выберите тип, соответствующий типу справочной информации.

Тип запроса поддержки | Когда следует использовать
--- | ---
Происшествие | Требовать управляемого рабочего стола операций группы разработчиков Microsoft для изучения, как широко распространены влияние изменений или безопасности происшествия.
Запрос сведений | Планировании изменений для сети, с прокси-сервером, VPN систем, истечение срока действия сертификата или достаточно некоторые сведения о службе. Ответ от управляемого рабочего стола операций группы разработчиков Microsoft настоятельно рекомендуется при взаимодействии изменения в вашей организации.
Запрос на изменение | Требовать управляемого рабочего стола операций группы разработчиков Microsoft для изменения, такие как перемещение звонков обновления устройств.

Дополнительные сведения:
- Поддержка языков — все поддержка предоставляется на английском языке.
- Изменения уровня серьезности - Microsoft могут перейти уровень серьезности, если клиент не может предоставить достаточное количество ресурсов или ответы для включения Майкрософт продолжить работу усилия решение проблемы. 
- Рабочие часы - для большинства стран, бизнес-часов: от 9:00:00 до 5:00, тихоокеанское время.
- Проблемы совместимости приложений, следует учитывать должен быть ошибку reproduceable той же версии приложения между предыдущей и текущей версией Windows или Office.
- Для устранения проблем совместимости приложений, корпорация Майкрософт требует точки клиента контактного лица для работы с. Отдельные необходимых для работы непосредственно с нашей командой Fast Track проанализируйте и устраните проблему.
- Клиент не может отвечать требованиям ожидаемый ответа, понизить версию Microsoft запроса по уровню один уровень серьезности, минимальный уровень серьезности C.
- Если клиент не отвечает на запросы, корпорация Майкрософт устранять и закрыть запрос поддержки в течение 48 часов последнего запроса.
