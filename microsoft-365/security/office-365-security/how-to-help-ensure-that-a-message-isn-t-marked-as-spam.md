# <a name="how-to-help-ensure-that-a-message-isnt-marked-as-spam"></a>Как помочь гарантировать, что сообщение не будет помечено как спам

Администратор Exchange Online или Exchange Online Protection (EOP) может исключить отнесение сообщений к спаму при передаче их через эту службу. Ознакомьтесь с обновленной версией советов и процедур для этого, чтобы [предотвратить пометку сообщений электронной почты как спама в Office 365](prevent-email-from-being-marked-as-spam.md).
